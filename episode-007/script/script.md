### 片段 01 [00:00-00:20] - 午后的院子

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室外/孤儿院院子 |
| 镜头 | 全景→中景 |
| 情绪 | 平静、空洞、孤独 |

**画面描写**:
[全景] 孤儿院院子。1994年的春天，阳光淡淡的，像一层薄纱。几棵枯树开始抽出新芽，嫩绿的，在灰色的枝条上颤抖。孩子们三三两两地待着——有的在玩耍，有的在发呆。

[中景] 陆鸣蹲在角落里。他的眼睛盯着地面，嘴唇在动——他在数地上的石头。一颗、两颗、三颗……

[特写] 陆鸣的手。他的手指在石头上轻轻划过，像在抚摸某种珍贵的东西。

**配音/台词**:
旁白: "那是1994年的春天。"

旁白: "孤儿院的午后，阳光很淡。"

旁白: "孩子们在院子里玩耍。"

旁白: "我蹲在角落里，数地上的石头。"

旁白: "一、二、三、四……"

**环境音**: 远处孩子的笑声，模糊的。风吹过树枝，发出细微的沙沙声。

**BGM**: 单音钢琴，平静的，像湖面的水波。

**歌词(竖排右上角)**:
自然的
馈赠

**画面动态标记**:
[静] 陆鸣蹲在角落，一动不动

### 片段 02 [00:20-01:00] - 数石头

| 属性 | 内容 |
|------|------|
| 时长 | 40秒 |
| 场景 | 室外/孤儿院院子 |
| 镜头 | 特写→中景 |
| 情绪 | 惊讶、发现、空洞 |

**画面描写**:
[特写] 地上的石头。大大小小，形状各异。陆鸣的眼睛在追踪每一颗石头，像在记录某种密码。

[特写] 陆鸣的眼睛。他的眼睛里有一种奇怪的光芒——不是悲伤，不是愤怒，是一种专注的、穿透一切的光芒。

[快切] 几颗石头的特写——
- 第一颗：灰色，椭圆形
- 第二颗：白色，有裂纹
- 第三颗：红色，很小

[特写] 陆鸣的嘴。他停止数数，闭上眼睛。

旁白: "我数石头的时候，发现了一件事。"

旁白: "我能瞬间记住很长的数字串。"

旁白: "比如：5839271460。"

旁白: "我看一遍，就记住了。"

旁白: "不是死记硬背，是自然而然地记住了。"

旁白: "像拍照一样。"

[中景] 一个老师路过。他看到陆鸣蹲在那里，嘴唇在动，停下了脚步。

老师: "你在干什么？"

陆鸣(睁开眼睛): "数石头。"

老师: "数到了多少？"

陆鸣: "37颗。"

老师(惊讶): "你数过一遍就记住了？"

陆鸣(点头): "5839271460。"

老师(困惑): "什么？"

陆鸣: "这是我刚才看的一串数字。随便想的。但我记住了。"

**配音/台词**:
旁白: "老师看着我，眼睛瞪得很大。"

旁白: "我不知道这有什么奇怪的。"

旁白: "这对我来说，就像呼吸一样自然。"

**环境音**: 风声，孩子们的远处笑声。

**BGM**: 钢琴声开始出现，有了一点旋律。

**歌词(竖排右上角)**:
拍照式
记忆

**画面动态标记**:
[动] 陆鸣闭上眼睛，嘴唇继续动
[静] 老师的表情从困惑变成惊讶

### 片段 05 [02:00-02:30] - 数字的本能

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/教室 |
| 镜头 | 中景→特写 |
| 情绪 | 发现、自信、困惑 |

**画面描写**:
[中景] 孤儿院教室。吴妈妈离开了，老师把陆鸣带到教室。

[特写] 黑板上的数学题：17 × 23 = ？

[特写] 陆鸣的脸。他的眼睛扫过那道题，眨了一下眼睛。

陆鸣(立刻): "391。"

[特写] 老师的表情。他正在演算，想确认答案是否正确。

老师(演算了几下，惊讶): "……对了。"

老师(又写下一道): "256 ÷ 8 = ?"

陆鸣(立刻): "32。"

[特写] 老师的脸。他的表情变得复杂——有惊讶，有佩服，还有一点点恐惧。

[特写] 陆鸣的眼睛。他的眼睛里有某种光芒，但也很空洞。

**配音/台词**:
旁白: "我发现我不只能记住数字。"

旁白: "我还能快速计算。"

旁白: "比如：17乘以23。"

旁白: "别人要列竖式，我要一瞬间。"

旁白: "答案是391。"

旁白: "我不知道为什么。"

旁白: "数字对我来说，像是一种本能。"

旁白: "像呼吸一样自然。"

旁白: "但有时候，这种本能让我害怕。"

旁白: "因为数字太多的时候，我会想起一些事情。"

**环境音**: 粉笔在黑板上划动的声音，老师翻纸的声音。

**BGM**: 钢琴渐强，有了一点自信的旋律。

**歌词(竖排右上角)**:
遗传的
诅咒

**画面动态标记**:
[动] 陆鸣的眼睛快速转动
[静] 老师的手停在半空中
