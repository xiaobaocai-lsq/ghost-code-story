### 片段 03 [00:50-01:30] - 外网扫描

| 属性 | 内容 |
|------|------|
| 时长 | 40秒 |
| 场景 | 室内/深渊据点 |
| 镜头 | 特写→中景→特写 |
| 情绪 | 专注、紧张、震惊 |

**画面描写**:
[特写] 陆鸣的屏幕。命令行窗口，nmap扫描结果在快速滚动。

[中景] 陆鸣坐在电脑前，手指飞快敲击键盘。猴子站在旁边看。

[特写] 陆鸣的眼睛。屏幕的光映在瞳孔里，快速移动。

[特写] 屏幕特写。扫描结果停在某一行：

"8080/tcp open  http-proxy Apache Struts 2.3.15"

[中景] 猴子的表情变了。他的眼睛瞪大了。

**配音/台词**:
猴子(压低声音): "你负责80和443。别的不用管。"

陆鸣(盯着屏幕): "……8080有个奇怪的banner。"

猴子(凑过来): "操，你运气不错。"

陆鸣(平静): "不是运气。是教科书第三章的案例。"

猴子(愣住了): "……什么？"

陆鸣: "Apache Struts 2.3.15，远程代码执行漏洞。CVE-2013-XXXX。"

猴子(倒吸一口冷气): "你他妈是新人？"

**环境音**: 键盘敲击声，服务器的嗡鸣

**BGM**: 节奏加快，音符变得密集
