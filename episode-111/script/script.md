**画面描写**:

[摇镜头-缓慢] 镜头扫过据点：有人在敲键盘，有人在低声交谈，屏幕的蓝光映在脸上。

[特写] 陆鸣的眼睛。深邃的，像两口井。他在思考什么。

**配音/台词**:

旁白: "他说，那个人会改变一切。"

**BGM**: 低沉的drone，电子脉冲，低频振动

|------|------|

| 镜头 | 全景→中景→特写 |
| 情绪 | 神秘、不安 |

**画面描写**:
[全景] 老季从门口走进来，身后跟着一个年轻女人。所有人都停下了手里的事，目光聚焦过来。

[中景] 女人穿着深灰色外套，黑色短发，皮肤苍白。她的步伐很稳，不紧不慢。

[特写] 裴婉的脸。冷静的眼神，几乎看不出情绪。她的嘴唇微微上扬，是一个礼貌的微笑。

[特写] 陆鸣的脸。他的眉头皱了一下——那一瞬间的表情变化，像是在回忆什么。

[中景] 老季拍了拍手，示意大家注意。

**配音/台词**:
老季(声音沙哑): "这是裴婉，代号'灰烬'。从今天起，她是我们的人。"

老季: "白屋派来的，专门负责这次行动的协调。"

旁白: "灰烬——燃烧后的残余。安静，冰冷，却藏着余温。"

裴婉(声音平静，轻微): "大家好。久仰各位的名字。"

裴婉(目光落在陆鸣身上): "你好，陆鸣。"

旁白: "她的声音很轻，像灰烬在风中飘散。"

旁白: "但她看我的眼神——像是看一个她认识很久的人。"

**环境音**: 窃窃私语声，椅子移动的声音

**BGM**: 钢琴声渐入，稀疏的音符，像犹豫的脚步

**歌词(竖排右上角)**:
灰烬降临
无声无息

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/陆鸣房间 + 电话 |
| 镜头 | 特写→中景 |
| 情绪 | 紧张、警告 |

**画面描写**:
[特写] 手机屏幕亮起。陈Sir的名字在屏幕上跳动。

[中景] 陆鸣靠在床头，接电话。房间很暗，只有手机的光。

[特写] 陆鸣的表情。眉头紧锁。

**配音/台词**:

陆鸣: "你怎么知道？"

陈Sir: "我有我的渠道。小心点，这种人不会无缘无故来。"

陆鸣: "你觉得她有问题？"

陈Sir(停顿): "我觉得她有目的。白屋的人，从来不做没有意义的事。"

陈Sir: "而且——"

陆鸣: "而且什么？"

陈Sir: "我查了她的背景。很干净。太干净了。"

陆鸣: "……"

陆鸣: "我知道。"

陈Sir: "你不知道。你不知道他们会怎么对付叛徒。"

旁白: "陈Sir的声音在发抖。"

旁白: "他很少这样。"
