### 片段 01 [00:00-00:20] - 任务结束后的沉默

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室内/深渊基地角落 |
| 镜头 | 特写→中景 |
| 情绪 | 疑惑、不安、颤抖 |

**关键帧规划**:
- 00:00-00:05: U盘特写，手指攥紧
- 00:05-00:10: 陆鸣侧脸，基地昏暗灯光
- 00:10-00:15: U盘插入电脑（主观）
- 00:15-00:20: 屏幕反光在陆鸣脸上

**画面描写**:
[特写] 一只手。指节发白，攥着一个黑色的U盘。U盘上有一道划痕——任务时留下的。

[中景] 陆鸣坐在基地角落的阴影里。身后是闪烁的服务器灯，像无数只眼睛在黑暗中眨动。

[特写] 陆鸣的脸。他的眼睛盯着电脑屏幕，瞳孔在屏幕光的映照下收缩。

[特写] 屏幕上是一份文件的标题——"长城公司案 1993"。但下面还有一行小字："陆远航 特别顾问"。

**配音/台词**:
旁白: "任务结束了。但我的手在抖。"

旁白: "目标家里有一台电脑，没有关机。我只是想看一眼。"

旁白: "我看到了'陆远航'三个字。"

**环境音**: 服务器风扇的嗡嗡声，偶尔的电流噼啪声。远处有脚步声，但很远。

**BGM**: 低频脉冲，若有若无，像远处的雷声。节奏是心跳的两倍慢。

**歌词(竖排右上角)**:
颤抖的手
握不住的真相

**画面动态标记**:
[动] 陆鸣的呼吸起伏，手指在U盘上摩挲，屏幕光在他脸上闪烁
