[特写] 陆鸣的脸。他的眼睛在屏幕上搜索着什么。

[特写] 苏晴的手。她打开了一个新的文件。

苏晴(看着屏幕，停顿了一下): "她……"

陆鸣: "怎么了？"

旁白: "屏幕上写着一行字——"

旁白: "那是我长大的地方。"

旁白: "她也去过那里。"

### 片段 03 [00:50-01:20] - 回忆涌现

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/情报室 + 闪回 |
| 镜头 | 特写→闪回→特写 |
| 情绪 | 震惊、困惑、回忆 |

**画面描写**:
[特写] 陆鸣的脸。他的表情变了——震惊，然后是困惑。

[主观视角-闪回] 模糊的画面。孤儿院的院子，一个小女孩蹲在墙角，低着头。陆鸣站在不远处看着她。

[特写] 陆鸣的手。他的手指在发抖。

[特写] 屏幕上的文字："1999年被家庭收养，离开孤儿院。"

[中景] 苏晴回头看着陆鸣，表情担忧。

**配音/台词**:
旁白: "向阳孤儿院。我以为我了解那里的每一个人。"

旁白: "但她的名字不在我的记忆里。"

旁白: "我漏掉了什么？"

苏晴: "她在向阳待到十岁。然后被一个家庭收养，离开了。"

苏晴: "但领养手续很奇怪。不是正常的家庭领养。"

陆鸣: "什么意思？"

苏晴: "是被一个组织带走的。手续上写的是'特别收养'。"

旁白: "被一个组织带走的。"

旁白: "就像我父亲出事之后，有人来带走我一样。"

苏晴: "之后的记录是空白的。像是被人抹掉了。"

旁白: "被人抹掉的过去。和我父亲的死一样。"

**环境音**: 电脑的嗡嗡声，回忆中的风声

**BGM**: 钢琴声渐入，带有困惑和不安

**歌词(竖排右上角)**:
记忆裂缝
拼图缺失

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/据点走廊 |
| 镜头 | 中景→特写→中景 |
| 情绪 | 试探、对峙、紧张 |

**画面描写**:
[中景] 据点的走廊。陆鸣站在裴婉面前，两人相对而立。

[特写] 陆鸣的眼神。警惕，带着质问。

[特写] 裴婉的脸。她的表情平静得出奇。

[特写] 裴婉的眼睛。她的眼睛里有一种奇怪的情绪——像是歉意，又像是别的什么。

**配音/台词**:
陆鸣: "你在向阳孤儿院待过。"

裴婉(眼神没有变): "你怎么知道的？"

陆鸣: "我也在那里长大。"

裴婉(停顿了一下): "……"

陆鸣: "你认识我吗？"

裴婉(缓缓开口): "我认识每一个从向阳出来的人。"

陆鸣: "你没有回答我的问题。"

裴婉(微笑，但那笑里有什么不对劲): "有些答案，你可能不想知道。"

陆鸣: "你什么意思？"

裴婉: "你是陆远航的儿子。"

旁白: "她的语气不是疑问，是陈述。"

旁白: "她早就知道我是谁。"

裴婉(轻声): "从一开始就知道。"

**BGM**: 钢琴声，紧张而悬疑

**歌词(竖排右上角)**:
四目相对
真相迷雾

| 属性 | 内容 |

**画面描写**:

[中景] 一辆黑色的车停在孤儿院门口。有人从车上下来。

[特写] 一个小女孩的背影。她穿着单薄的衣服，站在院子里，看着车。

[中景] 老院长站在她身边，表情复杂。

[特写] 小女孩回头。她看着孤儿院的方向——镜头缓慢推向她的脸。

[特写] 小女孩的眼睛。那眼神里有一种奇怪的平静，像是早就知道会发生什么。

**配音/台词**:
旁白(回忆): "1999年冬天。"

旁白: "有人来孤儿院带走了一个女孩。"

旁白: "她的名字不在我的记忆里——但她的眼神，我记得。"

旁白: "她走的时候，回头看了一眼。"

旁白: "那是她最后一次回头。"

旁白: "从那以后，没有人再提起她。"

旁白: "就像她从来没有存在过。"

旁白: "直到今天。"

**环境音**: 风声，雪花飘落的声音，回忆的空洞感

**BGM**: 钢琴独奏，悲伤而空灵

**歌词(竖排右上角)**:
