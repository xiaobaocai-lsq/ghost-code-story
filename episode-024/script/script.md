### 片段 01 [00:00-00:20] - 阿龙的邀请

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室内/网吧 |
| 镜头 | 中景 |
| 情绪 | 平静→好奇 |

**关键帧规划**:
- 00:00-00:05: 网吧日常
- 00:05-00:10: 阿龙出现
- 00:10-00:15: 阿龙说话
- 00:15-00:20: 陆鸣的反应

**画面描写**:
[中景] 畅通网吧。下午，客人不多。陆鸣在擦桌子，动作已经非常熟练。

[中景] 门口出现一个身影。阿龙。他靠在门框上，看着陆鸣。

阿龙: "有空吗？"

旁白: "一周后，阿龙又来了。"

[中景] 陆鸣抬头看了阿龙一眼，继续擦桌子。

阿龙: "有人想见你。"

[特写] 陆鸣的手。动作停了一下。

旁白: "我说，谁。"

阿龙: "叫Needle。"

阿龙: "他说，你通过了筛选。"

[特写] 陆鸣的脸。他的眼睛眯了一下。

旁白: "筛选？我什么时候被筛选过？"

阿龙: "你不知道的事多了。"

**配音/台词**:

- 00:55-01:05: Needle背影
- 01:05-01:15: Needle转身
- 01:15-01:20: 陆鸣的反应

**画面描写**:
[全景] 仓库内部。昏暗，只有几盏应急灯在闪烁。发电机发出嗡嗡的声响。角落里摆着几台电脑，屏幕的光在黑暗中格外刺眼。

[中景] 陆鸣的眼睛在暗中适应。他看到了一个人影，坐在电脑前，背对着门。

[特写] 那个人影。瘦削的背影，头发有些长，穿着深色的外套。

Needle(背对，声音平静): "你就是那个13岁的小孩？"

[特写] 陆鸣的脸。他没有说话，只是看着那个背影。

Needle(转过身): "阿龙说，你用test账号进了游戏服务器。"

[特写] Needle的脸。25岁左右，瘦削，戴着一副黑框眼镜，眼睛在镜片后面闪着光。

Needle: "那个漏洞是我故意留的。"

[特写] 陆鸣的眼睛。他的瞳孔缩了一下。

旁白: "我看着他。"

旁白: "故意的。"

旁白: "从一开始就是故意的。"

**配音/台词**:

旁白: "我看着他。故意的。从一开始就是故意的。

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/仓库 |
| 镜头 | 特写 |
| 情绪 | 专注、思考、紧张 |

**关键帧规划**:
- 01:50-02:00: 陆鸣观察文件
- 02:00-02:10: 思考
- 02:10-02:20: 输入命令

**画面描写**:
[中景] 陆鸣走向电脑。他的脚步很稳，没有犹豫。

[特写] 陆鸣的手。他把鼠标移到文件图标上，右键点击，查看属性。

[画面] 文件属性窗口：类型是Microsoft Word 97文档，加密方式是Word Default Encryption。

旁白: "我走近电脑。"

旁白: "我看了文件属性。加密方式是Microsoft Word 97加密。"

旁白: "那种加密很老，可以用工具破解。"

旁白: "但这里没有工具。"

旁白: "只有一台电脑，一个DOS命令行。"

[特写] 陆鸣的脸。他的眼睛在镜片后面快速转动，像在计算。

旁白: "我想了想，开始输入命令。"

[特写] 屏幕。DOS命令提示符在闪烁。

旁白: "Word 97的加密有个漏洞。"

旁白: "它有一个紧急修复功能，可以在不验证密码的情况下打开文件。"

旁白: "只需要一行命令。"

[特写] 陆鸣的手指在键盘上悬停。

旁白: "但我不能确定行不行。"

旁白: "这是测试。他们可能在看着我。"

旁白: "但我没有别的选择。"

**配音/台词**:
旁白: "我看了文件属性。加密方式是Microsoft Word 97加密。那种加密很老，可以用工具破解。但这里没有工具。只有一台电脑，一个DOS命令行。"
旁白: "Word 97的加密有个漏洞。它有一个紧急修复功能，可以在不验证密码的情况下打开文件。只需要一行命令。"

**环境音**: 键盘敲击声，心跳声

**BGM**: 节奏变得紧凑，像思考的节奏。

### 片段 06 [02:20-02:50] - 破解成功

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/仓库 |
| 镜头 | 特写→中景 |
| 情绪 | 紧张→成就感→惊讶 |

**关键帧规划**:
- 02:20-02:30: 输入命令
- 02:30-02:35: 文档打开
- 02:35-02:45: Needle反应
- 02:45-02:50: 陆鸣发现内容

**画面描写**:
[特写] 陆鸣的手指按下回车键。

[特写] 屏幕。DOS命令执行：

C:\>winword /tC:\test.doc /mEasyVistaToNewDoc

[画面] Word启动，文档在几秒后打开了。

旁白: "文档打开了。"

[特写] 陆鸣的脸。他的眼睛快速扫过屏幕上的文字。

[特写] 屏幕上的内容。是一封信：

"To whoever finds this:

If you're reading this, you've passed the test. Welcome to the Abyss.

- H"

旁白: "Needle看着屏幕，嘴角动了一下。"

[特写] Needle的脸。他在笑，但笑容很淡。

Needle: "有意思。"

[特写] 陆鸣的眼睛。他看着屏幕上的字，眼神复杂。

旁白: "欢迎来到深渊。"

旁白: "那是我第一次看到这个词。"

旁白: "深渊。"

**配音/台词**:
旁白: "文档打开了。"
屏幕内容: "To whoever finds this: If you're reading this, you've passed the test. Welcome to the Abyss. - H"
Needle: "有意思。"
旁白: "欢迎来到深渊。那是我第一次看到这个词。深渊。

**环境音**: 键盘声消失，只剩下发电机的嗡嗡声

**BGM**: 电子音突然变强，像打开了新世界的大门。然后是弦乐的轰鸣。

**歌词(竖排右上角)**:

深渊

看到这个词
