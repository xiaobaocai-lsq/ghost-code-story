### 片段 02 [00:15-00:45] - 找到韩默

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/基地走廊 |
| 镜头 | 中景→近景 |
| 情绪 | 紧张、愤怒、坚定 |

**关键帧规划**:
- 00:15-00:22: 走廊全景，韩默在前
- 00:22-00:28: 陆鸣快步追上
- 00:28-00:35: 陆鸣挡住韩默
- 00:35-00:45: 对峙

**画面描写**:
[全景] 基地的走廊。灯光昏暗，墙壁是灰色的。韩默的背影在走廊尽头，手里拿着一个文件夹。

[中景] 陆鸣从拐角走出。他的脚步很快，越来越快。

[近景] 陆鸣的脸。表情是坚定的，但眼睛里有东西在燃烧。

[中景] 陆鸣挡在韩默面前。韩默停下脚步，但没有抬头。

**配音/台词**:
陆鸣(声音压得很低，但每个字都像钉子): "师父。"

韩默(头也不抬): "让开。"

陆鸣(没有动): "不。"

**环境音**: 脚步声，陆鸣的呼吸声。

**BGM**: 低频脉冲变得明显，像心脏在加速。

### 片段 03 [00:45-01:30] - 对峙

| 属性 | 内容 |
|------|------|
| 时长 | 45秒 |
| 场景 | 室内/基地走廊角落 |
| 镜头 | 近景→特写→中景 |
| 情绪 | 愤怒、压抑、紧张 |

**关键帧规划**:
- 00:45-00:55: 韩默终于抬头
- 00:55-01:05: 陆鸣质问眼神
- 01:05-01:15: 韩默转身背对
- 01:15-01:30: 沉默对峙

**画面描写**:
[近景] 韩默终于抬起头。他的眼神是冷的，像在看一个陌生人。

[特写] 陆鸣的眼睛。他的眼睛里有愤怒，有受伤，还有一种奇怪的坚定。

[特写] 韩默的侧脸。灯光在他脸上形成阴影，遮住了半边脸。

[中景] 陆鸣把韩默堵在角落。他的姿态是攻击性的，但他的手在抖。

**配音/台词**:
陆鸣(声音发抖，但每个字都很清晰): "你早就知道我父亲的事。"

韩默(终于抬头，眼神变了一下，但很快恢复平静): "你想知道什么？"

陆鸣: "他是怎么死的。"

韩默(沉默很久，像是在压制什么): "他们说是自杀。"

陆鸣(冷笑): "你信吗？"

韩默(转身，背对陆鸣，声音变得很轻): "信不信有什么区别。"

**环境音**: 沉默。只剩下某种低频的嗡鸣。陆鸣压抑的呼吸声。

**BGM**: 弦乐开始出现，不和谐的，像有什么在被撕裂。
