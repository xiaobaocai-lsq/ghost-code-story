## 二、各Agent职责与产出规范

### 2.1 导演 (Director)

**核心职责**:
- 审核每一话的`话大纲`(Episode Outline)
- 确保人物形象一致性
- 审核分镜脚本(Storyboard Script)
- 最终成片质量把关

**审核节点**:
1. 话大纲审核 → 通过后进入编剧
2. 分镜脚本审核 → 通过后进入制作
3. 关键帧抽检 → 人物一致性
4. 最终成片审核 → 发布

---

### 2.2 编剧 (Writer)

**核心职责**:
- 根据话大纲，编写详细的分镜脚本
- 精确到每个时间点的场景描述
- 明确音效、BGM、角色台词
- 产出视频工作组和声音工作组的制作指令

**产出文件**: `episode-XX/script/script.md`

**脚本格式规范**:
```markdown
# 第X话 《标题》分镜脚本

## 基本信息
- 时长: ~3分钟 (180秒)
- 场景数: N个
- 主要人物: XXX, XXX
- 情绪基调: XXXX

## 分镜脚本

### 片段 01 [00:00-00:08]
| 属性 | 内容 |
|------|------|
| 场景 | 封面 |
| 镜头 | 全景 |
| 画面描述 | 黑暗中的雪夜，一扇窗户透出微弱的光 |
| 字幕 | 幽灵代码 / GHOST CODE |
| 音效 | 无 |
| BGM | 无 |
| 备注 | 建立氛围，定调 |

### 片段 02 [00:08-00:22]
| 属性 | 内容 |
|------|------|
| 场景 | 陆鸣房间 |
| 镜头 | 中景 → 特写 |
| 画面描述 | 8岁陆鸣坐在窗边，望着窗外... |
| 字幕/配音 | "1993年冬天，北方某小城。" (旁白) |
| 音效 | 风声、雪 |
| BGM | 渐入，氛围音乐 |
| 歌词(竖排右上角) | 雪落无声 / 埋藏真相 |
| 备注 | 回忆感 |

### 片段 03 [00:22-00:36]
...
```

### 2.3 视频工作组 (Video Team)

**核心职责**:
- 根据分镜脚本生成关键帧图片
- 维护人物一致性(使用参考图)
- 确保场景美术风格统一

**工作流程**:
1. 读取参考图(人物立绘、场景设定)
2. 按照脚本生成每个场景的关键帧
3. 使用参考图确保人物一致性
4. 输出图片到 `episode-XX/images/`

**关键帧命名规范**:
```
epXX_scene_01.jpg   # 场景1
epXX_scene_02.jpg   # 场景2
epXX_char_ref_luoming_8y.jpg  # 人物参考图
```

**一致性保障**:
- 同一人物在不同场景使用相同参考图seed
- 记录已使用的人物形象seed/fingerprint
- 重大形象变更需导演审核

---

### 2.4 声音工作组 (Audio Team)

**核心职责**:
- 生成角色配音(对话、独白)
- 生成BGM(氛围音乐)
- 生成音效(环境音、动作音)
- 多轨混音输出

**产出文件**:
```
episode-XX/audio/
├── voice_narrator_01.mp3    # 旁白1
├── voice_luoming_01.mp3      # 陆鸣配音
├── voice_father_01.mp3       # 父亲配音
├── sfx_door_slam.mp3         # 音效-门
├── sfx_car_passing.mp3       # 音效-车
├── bgm_scene_01.mp3         # BGM-场景1
├── bgm_full.mp3             # 完整BGM
├── mix_track.mp3            # 混音后音轨
```

**时间轴规范**:
```json
{
  "audio_timeline": [
    {"time": 0, "type": "silence", "duration": 8},
    {"time": 8, "type": "voice", "file": "voice_narrator_01.mp3", "lyrics": "1993年冬天，北方某小城。"},
    {"time": 22, "type": "voice", "file": "voice_luoming_01.mp3", "lyrics": "再过三天就是我八岁生日。"},
    {"time": 36, "type": "voice", "file": "voice_father_phone.mp3"},
    {"time": 68, "type": "bgm", "file": "bgm_tense.mp3", "volume": 0.5},
    {"time": 76, "type": "voice", "file": "voice_father_dying.mp3"},
    {"time": 124, "type": "bgm", "file": "bgm_police.mp3", "volume": 0.5},
    {"time": 144, "type": "voice", "file": "voice_narrator_final.mp3"}
  ]
}
```

### 2.5 合成组 (Compositing)

**核心职责**:
- 接收视频组的关键帧
- 接收声音组的音轨
- 使用ffmpeg合成视频
- 处理字幕叠加

**工作流程**:
1. 按时间轴拼接视频片段
2. 叠加多轨音频(配音+BGM+音效)
3. 烧录字幕到视频(竖排歌词)
4. 输出最终视频

**输出规范**:
```
episode-XX/output/
├── ghost_code_epXX.mp4       # 最终视频
├── ghost_code_epXX_720p.mp4 # 720p版本
├── ghost_code_epXX_1080p.mp4 # 1080p版本
```

---

### 2.6 字幕组 (Subtitle)

**核心职责**:
- 根据分镜脚本生成SRT字幕
- 处理歌词竖排显示(烧录到视频)
- 校对台词时间轴

**产出文件**:
```
episode-XX/subtitles/
├── dialogue.srt          # 对白字幕
├── lyrics.srt            # 歌词字幕
├── signs.srt             # 场景字幕/标题
```

## 四、工作流程详解

### 第一阶段: 策划期

```
1. 导演 → 确定话大纲 (Outline)
   - 根据STORY_OUTLINE.md确定本话内容
   - 确定出场人物、核心场景
   - 产出: episode-XX/outline.md

2. 导演审核 → 通过后进入制作
```

### 第二阶段: 制作期

```
3. 编剧 → 产出分镜脚本
   - 阅读话大纲
   - 编写详细的分镜脚本
   - 包含时间码、画面描述、音频指令
   - 产出: episode-XX/script/script.md

4. 导演审核 → 脚本通过
   - 检查脚本完整性
   - 检查时间轴合理性
   - 确认后进入制作

5. 并行制作:
   ┌─────────────────────────────────────────────┐
   │  视频组                    │  声音组         │
   │  - 生成关键帧             │  - 生成配音      │
   │  - 使用人物参考图         │  - 生成BGM       │
   │  - 保持美术风格统一       │  - 生成音效      │
   └─────────────────────────────────────────────┘

6. 字幕组 → 根据脚本生成字幕文件
```

### 第三阶段: 合成期

```
7. 合成组 → 合成视频
   - 拼接视频片段
   - 混合多轨音频
   - 烧录字幕(竖排歌词)
   - 产出: episode-XX/output/ghost_code_epXX.mp4

8. 导演终审 → 最终成片
   - 观看完整视频
   - 检查音画同步
   - 检查字幕正确性
   - 通过后发布
```

### 5.2 编剧Prompt

```
# 编剧 Agent

## 角色定位
你是《幽灵代码》的编剧，负责将话大纲转化为可执行的分镜脚本。

当BGM有歌词时，在"歌词(竖排右上角)"字段标注:

- 压抑、紧张、温暖、怀旧、暴力、悬疑

## 质量检查
- [ ] 每个片段都有时间码
- [ ] 配音和字幕匹配
- [ ] BGM/音效时间点精确
- [ ] 情绪转换平滑

## 七、错误处理机制

### 发现问题时的处理流程:

```
1. 发现问题 → 标记文件为 [有问题]
2. 记录问题详情
3. 通知上一环节负责人
4. 等待修改后重新提交
```

### 常见问题处理:

| 问题类型 | 处理方式 |
|---------|---------|
| 人物形象不一致 | 重新生成 + 使用参考图 |
| 配音与字幕不匹配 | 重新对齐时间轴 |
| BGM位置错误 | 调整时间轴 |
| 音画不同步 | 合成组重新混音 |
