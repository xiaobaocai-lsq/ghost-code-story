## 信仰演变史

### 1998-2005：理想主义黄金期

**阶段特征**：
- 成员多为理想主义黑客
- 真正相信"用技术改变世界"
- 主要任务：破解外国政府网站、传播被封锁的信息
- 对外形象："黑客罗宾汉"

**代表事件**：
- 1999年北约轰炸中国大使馆期间，组织参与对美网络攻击
- 2001年中美黑客大战，深渊成员活跃

**内部氛围**：
```
"我们在做正确的事，即使世界不理解"
```

---

### 2005-2015：变质期

**阶段特征**：
- 逐渐被白屋渗透
- 开始接"灰色订单"
- 理想派与务实派内斗
- 成员开始分化

**转折点**：
- 2005年，组织接到第一个"商业委托"
- 韩默在内部争论："我们的初心是什么？"
- 投票结果：接受订单，深渊开始变质

**内部氛围**：
```
"理想不能当饭吃，但也不能丢"
"我们可以保持初心，同时活下去"
```

---

### 2015-2020：分裂与清洗

**阶段特征**：
- 理想派试图"清党"
- 韩默开始秘密收集罪证
- 组织彻底沦为白屋工具
- 深渊名存实亡

**关键事件**：
- 2015年，理想派成员串联密谋
- 2016年，理想派成员"意外"死亡
- 2017年，韩默发现白屋已完全控制深渊
- 2018年，开始秘密备份证据

**内部氛围**：
```
"深渊已经死了，我们只是住在尸体里的幽灵"
"至少我们还活着，活着就能做点什么"
```

---

### 2020后：名存实亡

**阶段特征**：
- 只剩外壳
- 核心成员各怀鬼胎
- 白屋直接指挥行动
- 陆鸣的卧底行动在此背景下展开

## 成员入会仪式

**形式**：在IRC或加密聊天室进行

**流程**：
1. 申请者需完成一个"测试任务"
2. 通过后，获得一个临时代号
3. 三个月考核期内，只能接触外围信息
4. 考核通过后，正式获得深渊代号
5. 永远不知道其他成员的真实身份

**誓言**：
```
我以代码为证，在此宣誓：
我将守护深渊的秘密，像守护我的灵魂；
我将执行组织的任务，像执行我的意志；
在深渊，我无名；在深渊，我有家；
若有一日我背叛，愿我的代码永被诅咒。
—— 深渊成员誓言
```
