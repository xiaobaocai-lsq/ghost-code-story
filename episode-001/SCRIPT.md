# 第1话：衣柜

**章节**：第一幕·黑暗起源 - 第一章·灭门（1/10）

**时间**：1993年冬天

**核心事件**：8岁陆鸣目睹父亲被杀

**情绪基调**：压抑、恐惧、冰冷

---

## 第1页

### 画面
```
[远景 - 俯拍]
雪夜。北方小城边缘的老旧居民楼群。
楼间距狭窄，积雪覆盖着灰暗的屋顶。
远处零星的灯火，大部分窗户是黑的。
寒冷、空旷、孤独。

[字幕淡入]
「1993年冬天」
「北方某小城」
```

### 音效/氛围
```
环境音：风雪呼啸，远处偶尔有狗吠
色调：冷蓝灰，雪地的反光
```

---

## 第2页

### 画面
```
[中景 - 跟拍]
居民楼二楼，窗户透出昏黄的光。
窗帘拉着一半，能看到人影晃动。
窗户上的冰花在灯光下闪烁。

[特写]
窗台上摆着一盆枯萎的花。
花盆上贴着一张歪歪扭扭的纸条，写着"鸣鸣生日快乐"。

[字幕]
「再过三天，就是我八岁生日」
```

---

## 第3页

### 画面
```
[室内 - 客厅]
老式家具，笨重的电视机，墙上挂着日历。
一个男人背对镜头，坐在旧沙发上。
他在打电话，声音压得很低。

[特写 - 手]
男人的手夹着烟，蓝白色的烟圈缓缓升起。
手指修长，指节分明——和陆鸣的手一模一样。

[男人的声音 - 画外音]
"我说了，不该我管的事，我不管"
"他们给再多钱我也不接"
"你告诉老陆，让他自己小心点"
```

---

## 第4页

### 画面
```
[卧室门口 - 8岁陆鸣视角]
门缝中看出去，父亲的背影在客厅。
他站在走廊的阴影里，手里抱着一个旧玩偶——小熊。

[字幕]
「爸爸最近总是很晚回家」
「妈妈走了以后，他就不怎么说话了」

[特写 - 陆鸣的眼睛]
从门缝看出去的眼睛，
恐惧、好奇、还有一点点期待。
```

---

## 第5页

### 画面
```
[突然 - 音效：咣当一声]
门被踢开。
陆鸣本能地蹲下，躲进门后的衣柜。

[主观视角 - 衣柜缝隙]
模糊的世界。
几个黑影冲进来。

[音效]
脚步声、翻找声、父亲的声音：
"你们找错人了"
然后是一声闷响。
```

---

## 第6页

### 画面
```
[主观视角 - 颤抖]
从衣柜缝隙看出去。
父亲跪在地上，双手被反绑。
三个蒙面人站在周围。

[特写 - 父亲的脸]
父亲的眼神——不是恐惧，是一种奇怪的平静。
他的目光移向衣柜的方向。
看向陆鸣。

[父亲的嘴唇在动]
（无声，只有画面）
"不是我...是他们..."
```

---

## 第7页

### 画面
```
[特写 - 蒙面人的手]
其中一人手里有刀。
刀刃在昏暗的灯光下闪了一下。

[特写 - 父亲的眼睛]
父亲的眼睛闭上了。
嘴角似乎动了动，像是在说一个名字。

[音效]
一声沉闷的撞击。
然后是身体倒地的声音。
```

---

## 第8页

### 画面
```
[主观视角 - 衣柜缝隙]
陆鸣用手紧紧捂住嘴。
不让自己发出任何声音。
眼泪无声地流下来。

[特写]
柜门缝隙中，能看到父亲的手。
那只和陆鸣有着一样指节的手。
不再动了。

[音效]
寂静。
只有风雪声。
```

---

## 第9页

### 画面
```
[远景 - 俯拍]
居民楼外。
警车已经到达，红蓝灯闪烁。
警察进进出出。
楼下的积雪上已经被踩出一串脚印。

[特写 - 白布]
几个人抬着什么下楼。
白布下面，是人的形状。

[字幕]
「他们说是经济纠纷」
「他们说是爸爸犯了罪」
「没有人问我看到了什么」
```

---

## 第10页

### 画面
```
[最后一幕 - 黎明]
陆鸣被一个穿制服的人带下楼。
他回头看了一眼二楼。
窗户已经被封条封住。

[特写 - 陆鸣的脸]
八岁男孩的脸。
眼睛是干的。
没有眼泪。
从那以后，他再也没有哭过。

[字幕]
「那一年冬天」
「我失去了我的名字」

[黑屏]
```
