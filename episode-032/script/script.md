### 片段 01 [00:00-00:20] - 外围成员

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室内/深渊据点 |
| 镜头 | 中景→特写 |
| 情绪 | 好奇、紧张、专注 |

**画面描写**:
[中景] 陆鸣坐在据点角落的一台旧电脑前。屏幕的蓝光映在他脸上，显得他的黑眼圈更深了。

[特写] 电脑屏幕。一份文档打开着，标题是"深渊五大原则"。文档格式很简陋，像是从某个wiki直接复制下来的。

[特写] 陆鸣的眼睛。他的视线在屏幕上游移，但眉头微微皱起——他在思考，而不是单纯阅读。

[中景] 陆鸣的背影。他坐在一个塑料椅子上，椅子破旧，发出嘎吱声。工作台是用木板和砖头搭建的，但电脑很新。

**配音/台词**:
旁白: "成为深渊外围成员的第一周。我被分配到一个叫'猴子'的人手下。"

旁白: "他给了我一份文档。说这是深渊的信仰。"

旁白: "那天晚上，我一个字一个字地读。每一个字。"

**环境音**: 服务器的嗡鸣，空调的呼呼声，键盘敲击声

**BGM**: 低沉的电子脉冲，平稳，像学习时的心跳

**歌词(竖排右上角)**:
新的信仰
开始形成

### 片段 02 [00:20-00:50] - 第一原则：信息自由

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/深渊据点 |
| 镜头 | 中景→特写→中景 |
| 情绪 | 接受、思考、微妙质疑 |

**画面描写**:
[中景] 猴子站在陆鸣身后，双手抱胸，指着屏幕。他16岁，瘦小，眼睛灵活得像猴子——大概这就是他名字的由来。

[特写] 屏幕上的文字:

"第一原则：信息自由"
"1. 信息应该像空气一样自由"
"2. 知识不属于任何人"
"3. 防火墙是柏林墙"

[中景] 猴子的表情——带着一种传教士般的热情。

**配音/台词**:
猴子(得意): "这是我们最核心的信念。没有信息自由，就没有真正的民主。"

猴子: "政府捂着信息，像捂着黄金。防火墙，防的不是黑客，是人民。"

猴子: "我们的目标，就是让信息像水一样流动。挡不住的。"

[特写] 陆鸣的脸。他的表情是平静的，但眼神里有什么在转动。

**配音/台词**:
旁白: "听起来很崇高，对吧？"

旁白: "但我没有问——信息自由给谁带来了自由？"

旁白: "也没有问——泄露的信息，谁来负责？"

**环境音**: 服务器嗡鸣，猴子说话时的轻微回音

**BGM**: 钢琴声开始出现，稀疏的音符，像在思考

**歌词(竖排右上角)**:
信息应该像
空气一样自由

"1. 代码不会说谎"
"2. 代码不会背叛"
"3. 代码面前人人平等"

[中景] 猴子指着屏幕，越说越兴奋。

[特写] 陆鸣的手。他的手指无意识地在桌面上敲击，像在打字。

**配音/台词**:
猴子: "第二原则，代码即意志。"

猴子: "在代码的世界里，没有贫富差距，没有权力高低。"

猴子: "只有逻辑。一加一等于二，没有例外。"

猴子: "代码不会因为你是穷人就不给你服务。代码不会因为你是总统就给你开后门。"

[特写] 陆鸣的眼睛。他的眼神变了——那是被戳中记忆的神情。

**配音/台词**:
旁白: "我爸爸也这么想过。他相信代码是诚实的。"

旁白: "他曾经说，电脑是最诚实的东西。你输入什么，它就输出什么。"

旁白: "但那是他死之前说的话。"

旁白: "后来我才明白——代码是人写的。写代码的人有意志。"

旁白: "而意志，可以被操控。"

**环境音**: 键盘敲击声渐多，像某种共鸣

**BGM**: 钢琴声变得复杂，音符交错，像思绪

**歌词(竖排右上角)**:
代码不会说谎
但人会

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/深渊据点 |
| 镜头 | 特写→中景→特写 |
| 情绪 | 警觉、微妙紧张 |

**画面描写**:
[特写] 屏幕滚动:

"第四原则：匿名"
"1. 没有人知道你是谁"
"2. 成员之间以代号相称"
"3. 真实身份永远保密"

[中景] 猴子指着屏幕，表情变得严肃。

[特写] 屏幕继续滚动:

"第五原则：去中心化"
"1. 没有主服务器"
"2. 没有单点故障"
"3. 每个节点都同等重要"

**配音/台词**:
猴子: "第四，匿名。没有人知道你是谁，包括我们。"

猴子: "你在深渊的名字，只是名字。你的真实身份，只有你自己知道。"

猴子: "这是保护你的最好方式。"

[特写] 陆鸣的眼睛。

**配音/台词**:
猴子: "第五，去中心化。没有主服务器，没有单点故障。"

猴子: "就算干掉我们一半的人，剩下的人还能继续运转。"

猴子: "这是我们比那些大公司、大政府更强大的地方。"

[特写] 陆鸣的脸。他似乎想到了什么。

**配音/台词**:
旁白: "除了那个投资人？"

猴子(表情僵了一下): "……那是例外。"

旁白: "我问的是去中心化。"

旁白: "但他回答的是匿名。"

旁白: "他没有回答我的问题。从那一刻起，我知道——有些问题是不能问的。"

**环境音**: 服务器嗡鸣变大了，像是某种回应

**BGM**: 低频bass重新出现，保持压迫感

**歌词(竖排右上角)**:
例外
永远存在

### 片段 06 [02:20-02:50] - 裂缝

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/孤儿院宿舍 |
| 镜头 | 中景→特写→中景 |
| 情绪 | 思考、质疑、失眠 |

**画面描写**:
[全景] 孤儿院宿舍。深夜。月光从窗帘缝隙透进来，在地上投下一道白色的光。

[特写] 陆鸣躺在床板上，眼睛望着天花板。天花板上有水渍，形成各种不规则的形状。

[特写] 陆鸣的手。他在空中比划着什么——像是在敲键盘。

**配音/台词**:
旁白: "那天晚上，我在孤儿院的床上想这些原则。"

旁白: "每一条听起来都很完美。但每一条都有裂缝。"

旁白: "信息自由——但我们泄露的信息，谁来负责？"

旁白: "代码即意志——但如果写代码的人有恶意呢？"

旁白: "效率至上——但用什么标准衡量效率？谁的标准？"

旁白: "匿名——但匿名不是保护作恶者吗？"

旁白: "去中心化——但那个投资人不是中心吗？"

[特写] 陆鸣的眼睛。月光照在他脸上，眼睛像两口深井。

旁白: "我没有把这些问题说出来。"

旁白: "在孤儿院八年，我学会了一件事——问太多问题的孩子，会被孤立。"

旁白: "在深渊，大概也一样。"

**环境音**: 夜风从窗缝钻进来，发出呜咽声。远处有火车经过，汽笛声呜咽。

**BGM**: 钢琴声变得缓慢，像失眠者的叹息。音符稀疏，像星星。

**歌词(竖排右上角)**:
每一道裂缝
都是深渊
