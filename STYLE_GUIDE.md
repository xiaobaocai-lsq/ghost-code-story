# 《幽灵代码》视觉风格指南 (Style Guide)

---

## 一、整体视觉定位

**视觉主题**：赛博朋克 × 90年代复古 × 悬疑冷峻

**色彩基调**：
- 主色调：深蓝(#0a1628)、炭黑(#1a1a2e)
- 辅助色：霓虹蓝(#00d4ff)、霓虹紫(#7b2cbf)
- 强调色：暗红(#8b0000)、暖黄(#ffd700)
- 情绪色：灰白(#e0e0e0)用于希望场景

**场景基调**：
- 90年代中国城镇：褪色招牌、昏黄路灯、水泥墙
- 网络/数字世界：代码雨、数据流、扫描线
- "深渊"空间：全息投影、终端界面、黑暗中的蓝光

## 二、主角形象规范

### 陆鸣（幽灵）- 核心形象

#### 8岁版本
```
外貌：瘦小，近视眼（戴厚眼镜），眼神惊恐又倔强
发型：乱糟糟的短发
肤色：苍白
表情：极少，眼眶发红但不哭
衣着：旧棉袄，袖口磨破，膝盖有补丁
标志：左手腕上系着一根红绳（父亲给的）
```

#### 13-17岁版本（深渊时期）
```
外貌：苍白高瘦，近视（戴黑框眼镜），消瘦
发型：凌乱短发，刘海遮眼
肤色：苍白，几乎不见阳光
表情：冷漠，嘴角紧闭，偶尔冷笑
衣着：大一号的黑灰色运动服，兜帽永远拉下
标志：左眉有一道旧疤（孤儿院打架留下）
习惯动作：思考时用食指敲击桌面
```

#### 17-25岁版本（幽灵时期）
```
外貌：消瘦但结实，五官冷峻，目光锐利
发型：短发，鬓角干净利落
肤色：长期熬夜的灰白
表情：警惕、审视、冷静
衣着：深色外套，黑色T恤，连帽衫
标志：左眉的疤更明显，永远戴着一副旧耳机
习惯动作：紧张时摸左眉的疤
```

#### 25岁版本（结局）
```
外貌：憔悴但眼神坚定，面容比实际年龄老
表情：平静，带着淡淡的笑意
衣着：简洁白衬衫，袖子卷起
标志：左手腕的红绳还在
```

## 三、关键配角形象

### 苏晴
```
外貌：短发干练，五官清晰，眼神清澈坚定
年龄：与陆鸣相仿或略大
发型：齐耳短发或干练马尾
表情：直接，不懂得拐弯
衣着：警服或职业装，早期偏正式
标志：左手无名指有一道旧伤疤（父亲教训时留下）
```

### 韩默（棋手）
```
外貌：45岁看起来像35，消瘦儒雅
发型：花白短发，梳得一丝不苟
衣着：永远西装革履，但领带从不系紧
表情：温和的笑容，眼底有深深的疲惫
标志：修长的手指，弹钢琴般优雅
习惯：下棋时喜欢闭眼思考
```

### 裴婉（灰烬）
```
外貌：冷艳，高挑，眼神空洞
发型：黑色长发，或扎或散
表情：极度冷静，非必要不说话
衣着：黑色系，职业杀手风格
标志：左手腕有一道旧伤疤（入会仪式）
```

### 老黑
```
外貌：壮实，络腮胡，眼神沧桑
年龄：比陆鸣大约10岁
表情：沉默，但一笑很温暖
衣着：旧皮夹克，洗到发白的牛仔裤
标志：左手小指缺一节（工伤）
```

## 四、场景视觉规范

### 孤儿院
```
色调：灰黄为主，压抑
元素：斑驳的水泥墙、铁栏杆床、生锈的窗户
光线：永远是阴天或傍晚的灰光
象征物：院子里一棵枯树
```

### 网吧（90年代）
```
色调：昏黄烟雾缭绕
元素：CRT显示器、游戏海报、泡面盒
光线：屏幕的蓝光映在脸上
代表：墙上挂钟、嘎吱作响的椅子
```

### "深渊"服务器房间
```
色调：深蓝黑暗中一点蓝光
元素：服务器机柜、冷却风扇、数据灯闪烁
光线：幽蓝的LED，代码雨投影在墙上
声音：风扇的嗡嗡声
```

### 审讯室
```
色调：冷白荧光灯
元素：金属桌椅、单向玻璃、摄像头
光线：刺眼的顶灯，无阴影
```

## 六、镜头语言规范

### 分镜基本规则

```
[远景] - 建立场景，通常用于话首
[全景] - 展示人物与环境关系
[中景] - 人物对话/互动
[特写] - 强调情绪，关键时刻
[大特写] - 极端情绪，眼神
[主观镜头] - 角色视角，增强代入感
```

### 情绪镜头规范

| 情绪 | 镜头倾向 | 示例 |
|------|----------|------|
| **压抑** | 低角度仰拍，压在画面上方 | 孤儿院走廊 |
| **力量** | 高角度俯拍，显得渺小 | 韩默出场 |
| **亲密** | 浅景深，焦点在眼睛 | 陆鸣×苏晴 |
| **紧张** | 快速切换，碎片化 | 追逐戏 |
| **回忆** | 高对比黑白，柔焦边缘 | 闪回镜头 |

## 八、内容表达规范

### 暴力场景尺度
```
原则：暗示优于展示，声音优于画面
灭门场景：用衣柜缝隙视角、影子、声音暗示
死亡场景：用fade out、情绪特写代替直接表现
流血场景：最小化，可用暗红色代替
```

### 对话气泡规范
```
普通对话：白色气泡，黑字
内心独白：灰色背景，虚线边框
黑客对话/代码：绿色等宽字体气泡
危险暗示：红色边框
```

### 文字规范
```
标题字体：无衬线 Bold
正文字体：清晰易读
代码/黑客界面：等宽字体
时间戳：泛黄老旧字体
```
