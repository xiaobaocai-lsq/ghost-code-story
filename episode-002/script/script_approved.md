### 片段 01 [00:00-00:20] - 公交车上的告别

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 车内/行驶中 |
| 镜头 | 特写→远景 |
| 情绪 | 失去、迷茫、冰冷 |

**画面描写**:
[特写-车窗] 陆鸣的脸贴在车窗上，呼出的气在玻璃上结成雾。他用手指在雾上画了一个圆——圆里，是一个歪歪扭扭的"鸣"字。和那晚在父亲窗户上画的一样。但这一次，没有人从另一边回应他。

[中景] 公交车在灰暗的公路上行驶。窗外是倒退的杨树，光秃秃的枝丫像伸向天空的手指。雪已经化了，只剩下泥泞和灰色。

[远景] 公交车越来越小，消失在灰白的天际线里。

[特写] 陆鸣的眼睛。空洞的，干涩的，像被冻住的湖面。从那以后，他的眼睛就一直是这个样子。

**配音/台词**:
旁白: "他们把我塞进一辆公交车。那是1993年的最后一天。再过几个小时，就是1994年了。"

旁白: "车窗外的风景在倒退。我不知道这辆车要带我去哪里。我没有问。"

旁白: "我没有哭。不是因为不想，是因为眼泪在那个衣柜里，已经全部流干了。"

**环境音**: 公交车引擎的轰鸣声，沉重的，喘息般的。偶尔有车轮碾过碎石的嘎吱声。远处火车的汽笛声，呜咽般的。

**BGM**: 低沉的钢琴单音，在一个音符上反复，像心脏在停止前最后的跳动。

**歌词(竖排右上角)**:
雪落无声
记忆冻结

**画面动态标记**:
[动] 车窗上的雾气不断形成又消散，陆鸣手指的轨迹，车窗外倒退的风景

### 片段 02 [00:20-00:50] - 孤儿院大门

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室外/孤儿院门口 |
| 镜头 | 远景→中景→特写 |
| 情绪 | 压抑、陌生、恐惧 |

**画面描写**:
[远景] 公交车停在郊区的一个大院落前。灰墙斑驳，像一张苍老的脸。墙头的铁丝网在阳光下闪烁。

[摇镜头-右移] 镜头扫过灰墙上的字迹——"向阳孤儿院"。字迹已经褪色，有些笔画已经脱落。

[特写-铁门] 铁门生锈了，锈迹像干涸的血迹。门楣上有一个褪色的红十字。

[中景] 老院长站在门口。她60岁左右，头发花白，梳成一个髻。她的脸上有深深的皱纹，像老树皮。她的眼睛——那双眼睛在陆鸣脸上停留了一下。

[特写] 老院长的眼睛。那一瞬间的停顿，像是在看一个她认识很久的人。然后她笑了，但那笑容没有到达眼睛。

**配音/台词**:
旁白: "孤儿院院长姓吴。其他人叫她吴妈妈，但她不喜欢这个称呼。"

旁白: "她看我的时候，停顿了一下。就那么一下。"

旁白: "她看我的眼神很奇怪。像是在看一个她认识很久的人。"

旁白: "但我不认识她。我从来没有见过她。"

**环境音**: 风声从院子里传来，像是孩子们的哭声。远处有乌鸦在叫，孤零零的。

**BGM**: 低频drone开始出现，没有旋律，只有压抑的频率。

**歌词(竖排右上角)**:
新的牢笼
无人知晓

**画面动态标记**:
[动] 铁门在风中发出吱呀声，老院长的衣角被风吹动，她的眼神在陆鸣脸上停留

### 片段 03 [00:50-01:30] - 第一夜

| 属性 | 内容 |
|------|------|
| 时长 | 40秒 |
| 场景 | 室内/宿舍 |
| 镜头 | 全景→特写 |
| 情绪 | 孤独、恐惧、冰冷 |

**画面描写**:
[全景] 宿舍很小，挤着四张床。床板硬邦邦的，被褥单薄，散发着潮湿的霉味。墙角有一块脱落的墙皮，像一块伤疤。

[特写] 陆鸣缩在最靠墙的角落里。他的背包抱在胸口，像抱着最后一件属于他的东西。他的眼睛睁着，望着天花板。

[中景] 其他三个孩子已经睡着了。一个小女孩抱着一个破旧的布娃娃，嘴唇在动，像在梦呓。另一个小胖子在打呼噜。第三个孩子背对着所有人，蜷成一团。

[特写] 陆鸣的手。他的手指在背包的拉链上反复摩挲，发出细微的金属声。

[特写] 陆鸣的眼睛。天花板上有水渍，形成各种形状。他用眼睛追踪那些形状，试图把它们变成数字——一、二、三……

**配音/台词**:
旁白: "第一夜，我没有睡。"

旁白: "我躺在硬邦邦的床板上，听着其他孩子的呼吸声。那些声音像是从很远的地方传来。"

旁白: "爸爸以前说，睡不着的时候就数羊。但我数来数去，数出来的都是他的脸。"

旁白: "一、爸爸的脸。二、爸爸的手。三、爸爸的眼睛。四、爸爸的嘴——他在说什么？"

旁白: "我想不起来他最后说了什么。我只记得他的嘴在动。"

**环境音**: 风声呜咽，从窗缝钻进来。孩子的梦呓，模糊不清的。远处火车的汽笛声，呜咽般的。

**BGM**: 几乎无声，只有极低频的drone，像来自深渊的呼吸。

**歌词(竖排右上角)**:
黑暗中的
数字

**画面动态标记**:
[动] 窗帘在风中飘动，月光从窗帘缝隙透进来，在地上投下摇晃的影子

### 片段 04 [01:30-02:00] - 老三的话

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室外/院子角落 |
| 镜头 | 中景→特写 |
| 情绪 | 困惑、紧张、神秘 |

**画面描写**:
[中景] 第二天早上。陆鸣蹲在院子的角落里。阳光很淡，照在地上形成斑驳的影子。院子中央有一棵枯树，枝丫光秃秃的。

[特写] 老三走过来。他14岁，瘦高，沉默。他的眼睛在阴影里，像是两口深井。

[中景] 老三蹲下来，和陆鸣保持一段距离。他的影子在地上拉得很长，像一条黑色的绳子。

[特写] 老三的嘴唇在动，声音沙哑，像很久没有说过话。

**配音/台词**:
老三(声音沙哑): "你是陆家的小孩？"

旁白: "老三叫马超。他是孤儿院里待得最久的孩子。"

老三: "你爸姓陆，对吧？搞电脑的？"

老三(压低声音): "我劝你，别太打听这里的事。这里不正常。"

老三: "吴妈妈对每个孩子都好。但她总是在深夜烧东西。"

老三(眼神闪烁): "我亲眼看见过——她烧的是信。信封上有字，但我认不全。"

[特写] 老三的眼睛。他的眼睛里有一种警告的意味。

老三: "你爸的事……别查。"

**环境音**: 院子里有孩子在远处玩耍的声音，模糊的，遥远的。风声，像是有人在叹息。

**BGM**: 钢琴声开始出现，音符稀疏，像犹豫的脚步。

**歌词(竖排右上角)**:
沉默的警告
纸灰飞扬

**画面动态标记**:
[动] 风吹动枯树的枝丫，老三的影子在地上摇晃

### 片段 05 [02:00-02:30] - 发现电脑

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/电脑室 |
| 镜头 | 中景→特写→特写 |
| 情绪 | 发现、希望、专注 |

**画面描写**:
[全景] 孤儿院后面有一间小屋子。门半开着，里面透出绿色的光。

[中景] 陆鸣推开门。屋子里有一台老旧的电脑——386，绿色显示器，键盘上的字母已经磨掉了大半。

[特写] 陆鸣的眼睛。他的眼睛在看到显示器的瞬间亮了一下——那是他在父亲去世后第一次有了这种眼神。

[特写] 电脑屏幕。绿色的字符在黑色的背景上闪烁，像某种神秘的密码。

[特写] 陆鸣的手。他的手指悬停在键盘上方，像钢琴家在演奏前的停顿。

[特写] 陆鸣的脸。他的表情变了——不再是空洞的，而是专注的，像进入了一个只有他和电脑的世界。

**配音/台词**:
旁白: "1994年，中国接入了互联网。但对我来说，那台电脑意味着别的东西。"

旁白: "爸爸以前说，电脑是最诚实的东西。你输入什么，它就输出什么。它不会骗你。"

旁白: "它是冷的，但它不会背叛你。"

旁白: "我第一次破解了一个游戏。不是为了玩——是因为我想知道它是怎么工作的。"

旁白: "那个瞬间，我忘记了一切。忘记了爸爸，忘记了衣柜，忘记了眼泪。"

旁白: "只剩下我和它。"

**环境音**: 电脑风扇的嗡嗡声，键盘的咔哒声，显示器的电流声。

**BGM**: 钢琴声变得轻快，有了旋律，像泉水从地下涌出。

**歌词(竖排右上角)**:
数字的世界
唯一的语言

**画面动态标记**:
[动] 电脑屏幕的字符闪烁，陆鸣的手指在键盘上跳动，显示器的绿光在他脸上闪烁

### 片段 07 [03:00-03:20] - 吴妈妈的秘密

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室外/院子 |
| 镜头 | 主观视角→特写 |
| 情绪 | 恐惧、困惑、神秘 |

**画面描写**:
[主观视角] 深夜。月光很淡。陆鸣起来上厕所。

[中景] 他看见吴妈妈在院子角落里烧东西。火光照亮她的脸——那表情不是悲伤，是恐惧。

[特写] 吴妈妈的手。颤抖的，苍老的，像枯树枝。她的手里攥着一叠纸。

[特写] 火堆。纸张在火焰中卷曲、变黑、化为灰烬。灰烬飞舞，像灰色的蝴蝶。

[中景] 吴妈妈抬头。她看见了陆鸣——但她没有惊讶。她的嘴唇动了动。

[特写] 吴妈妈的脸。火光在她眼睛里跳跃。那眼神里有恐惧，有歉意，还有一种奇怪的解脱。

**配音/台词**:
旁白: "有一天深夜，我看见吴妈妈在烧东西。"

旁白: "火光映在她脸上。那不是悲伤的表情——是恐惧。"

旁白: "她看见了我。她的嘴唇动了动。"

旁白: "她说:'睡吧，孩子。'"

旁白: "她的声音在发抖。"

旁白: "她烧的是什么？那些纸上写了什么？"

旁白: "我开始用数字记录她。"

**环境音**: 火堆的噼啪声，纸张燃烧的嘶嘶声，远处夜鸟的叫声

**BGM**: 所有音乐停止，只剩下火焰的噼啪声——然后是寂静

**歌词(竖排右上角)**:
火焰中的
秘密

**画面动态标记**:
[动] 火焰在风中摇曳，灰烬飞舞，月光在地上晃动
