### 片段 01 [00:00-00:20] - 监控画面

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室内/网络安全部门监控室 |
| 镜头 | 远景→特写 |
| 情绪 | 紧张、压迫、冷酷 |

**画面描写**:
[远景] 网络安全部门监控室。巨大的屏幕墙闪烁着无数数据流，蓝白色的光在黑暗中跳动。服务器的风扇嗡嗡作响，像某种巨兽的呼吸。

[中景] 几个穿制服的技术人员在各自的屏幕前工作，表情专注而冷漠。

[特写-屏幕] 某个IP地址被红框标记。屏幕上跳动着"追踪中..."的字样。

[特写-档案] 屏幕上弹出一张照片——17岁陆鸣的脸，稚嫩但眼神锋利。旁边标注着："代号：幽灵 / 组织：深渊 / 危险等级：A"

**配音/台词**:
旁白: "1999年秋天，我以为自己是一把刀。"

旁白: "我不知道，有人已经在磨刀了。"

**环境音**: 服务器风扇的嗡嗡声，键盘的敲击声，监控设备的电子蜂鸣

**BGM**: 低沉的电子脉冲，像心脏在压迫中跳动。频率极低，像来自深渊的呼唤。

**歌词(竖排右上角)**:
网已张开
猎物入局

### 片段 02 [00:20-00:50] - 网吧里的少年

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/昏暗网吧 |
| 镜头 | 中景→特写→中景 |
| 情绪 | 专注、冷酷、孤独 |

**画面描写**:
[全景] 昏暗的网吧。烟雾缭绕，霓虹灯从窗外透进来，在天花板上投下斑驳的色彩。十几台电脑前坐着年轻人，他们沉浸在自己的世界里。

[特写-角落] 陆鸣坐在最角落的位置。他的脸被屏幕的光芒照亮，忽明忽暗。屏幕上滚动着代码，像某种神秘的咒语。

[特写-手] 陆鸣的手指在键盘上飞舞，速度快得几乎看不清。每一个按键都精准无误。

[特写-眼] 陆鸣的眼睛。屏幕的光映在他的瞳孔里，锐利的，专注的，和年龄完全不符的成熟。

**配音/台词**:
旁白: "那时候我在深渊已经待了两年。韩默说我是天才。"

旁白: "他不知道，我只是在寻找答案。"

旁白: "爸爸是怎么死的？那些蒙面人是谁？"

旁白: "我以为我能找到答案。"

**环境音**: 网吧的背景音乐，模糊的电子乐。键盘敲击声，此起彼伏。有人在抽烟，咳嗽声。

**BGM**: 电子节拍开始出现，节奏紧凑，像陆鸣思维的节奏。

**歌词(竖排右上角)**:
键盘为刀
代码为剑

| 镜头 | 中景→特写→中景 |
| 情绪 | 压抑、冷酷、紧张 |

**画面描写**:
[中景] 监控室内。陈Sir站在屏幕墙前，50岁左右，花白的头发梳得一丝不苟。他的背影像一座雕塑，一动不动。

[特写-屏幕] 屏幕上显示着陆鸣正在攻击的服务器地址。攻击路径被红线标注，像一张蜘蛛网。

[中景] 副手走过来，递上一份档案。档案封面上写着："重点监控对象"。

[特写-档案] 档案翻开。第一页是陆鸣的照片。第二页是陆鸣的个人信息。第三页——

[特写] "父亲：陆远航 / 死亡日期：1993年12月26日 / 案件状态：已结案（畏罪自杀）"

[特写] "相关调查已被封存"

[中景] 陈Sir接过档案。他的手在"陆远航"三个字上停留了一下。

**配音/台词**:

旁白: "他们说，我的技术太年轻，太张扬。"

旁白: "他们说，这个孩子，必须被找到。"

旁白: "他们没有说，他们认识我爸爸。"

**环境音**: 监控设备的电子蜂鸣，空调的嗡嗡声，翻动纸张的声音

**BGM**: 低频脉冲继续，节奏变得更有压迫感。

**歌词(竖排右上角)**:
