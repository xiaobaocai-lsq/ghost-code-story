### 片段 01 [00:00-00:20] - 网吧的午夜

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室内/地下网吧 |
| 镜头 | 特写→中景 |
| 情绪 | 孤独、冷淡、专注 |

**画面描写**:
[特写-电脑屏幕] 2008年，秋，凌晨2点。电脑屏幕的蓝光映在陆鸣脸上。屏幕上是一串代码和某个政府网站的后台界面。

[特写] 陆鸣的手。手指在键盘上飞快跳动，像蜘蛛在织网。他的眼睛——14岁却已经有黑眼圈的眼睛——盯着屏幕，专注得像在凝视深渊。

[中景] 地下网吧的全貌。十几台破旧的电脑，大部分都黑着屏。只有角落里几台还亮着，光线昏暗，空气浑浊，弥漫着烟味和泡面味。

[特写-屏幕] 陆鸣按下最后一个回车键。屏幕上弹出绿色字符："渗透成功"。同一秒，一个弹窗从屏幕右下角跳出来——

弹窗内容："join us"
只有一个按钮："接受"

[特写] 陆鸣的眼睛。他盯着那个弹窗，瞳孔微微收缩。

**配音/台词**:
旁白: "2008年秋天，我14岁。在孤儿院的第八年。"

旁白: "那一年，我遇到了改变我一生的人。或者说——改变我一生的东西。"

旁白: "那天晚上，我刚刚完成了一次'练手'。某政府网站的漏洞测试。很简单的作业。"

旁白: "但那个弹窗改变了一切。"

**环境音**: 电脑风扇的嗡嗡声，远处街道的夜车声，日光灯管的电流嗡鸣

**BGM**: 低沉的drone，像来自深海的回声。一个单音的电子脉冲，每隔几秒响一次，像心跳。

**歌词(竖排右上角)**:
黑暗中的
召唤

**画面动态标记**:
[动] 屏幕代码滚动，弹窗闪烁，陆鸣的呼吸

### 片段 02 [00:20-00:50] - 韩默的出现

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/地下网吧 |
| 镜头 | 主观视角→中景→特写 |
| 情绪 | 神秘、紧张、毛骨悚然 |

**画面描写**:
[主观视角-陆鸣] 他感觉到有人站在身后。网吧的日光灯管在头顶嗡嗡响，他的影子被拉得很长。

[中景] 陆鸣转头。一个黑色身影站在他身后三步远的地方。黑色连帽衫，帽子压得很低，只露出下半张脸。

[特写] 一只手——修长、苍白、指节分明——伸过来，把一张折叠的纸条放在陆鸣键盘旁边。

[中景] 陆鸣犹豫了一下，拿起纸条。展开。上面只有一个词：

纸条特写："深渊"

[特写] 陆鸣的眼睛。他的瞳孔收缩了一下，但表情没有变。八年孤儿院生活教会他一件事——永远不要在脸上显示情绪。

[中景] 黑影走近一步。日光灯的光照亮了他的下半张脸——下巴棱角分明，嘴唇抿成一条线。

**配音/台词**:
韩默(声音低沉，像砂纸摩擦): "你的技术不错。但你在用玩具刀打架。"

陆鸣(声音很轻，平静得不像14岁): "……你是谁？"

韩默(嘴角似乎勾了一下): "一个比你更老的人。"

**环境音**: 日光灯的电流声更大了，像某种警告。远处传来摩托车引擎的轰鸣声。

**BGM**: 低频bass开始出现，没有旋律，只有压迫感。节奏像脚步，一下一下。

**歌词(竖排右上角)**:
玩具刀
遇到真剑

**画面动态标记**:
[动] 纸条在键盘上微微振动（空调风），韩默的影子在地上拉长

### 片段 03 [00:50-01:30] - 地下据点

| 属性 | 内容 |
|------|------|
| 时长 | 40秒 |
| 场景 | 室内/废弃工厂深处 |
| 镜头 | 远景→中景→特写 |
| 情绪 | 压抑、科幻、震撼 |

**画面描写**:
[远景] 废弃工厂的外景。凌晨的秋夜，冷风从破碎的窗户灌进来。工厂的烟囱像一根根枯骨伸向天空。

[中景] 韩默走在前面，陆鸣跟在后面。穿过锈迹斑斑的走廊，墙上涂着八十年代的标语，红字已经褪色。

[中景] 一扇铁门。韩默从口袋里掏出一把钥匙，插入锁孔。咔哒一声，锁开了。

[主观视角-陆鸣] 门被推开一条缝。光线从里面透出来——不是日光灯的白光，是服务器指示灯的绿色荧光。

[中景] 韩默推开门，侧身让陆鸣进去。

[全景] 机房内部。几十台服务器在黑暗中运行，绿色的指示灯像满天星斗。十几个人坐在各自的工作台前，面对屏幕，没人抬头看一眼。

[特写] 服务器机柜。风扇在转动，散热口冒着热气，像某种巨兽的呼吸。

[中景] 陆鸣站在门口，他的脸上映着绿光。他的眼睛在扫视每一个角落——摄像头的位置，服务器的型号，人员的站位。他在评估。

**配音/台词**:
旁白: "韩默把我带到一个地方。废弃工厂，听起来像电影。"

旁白: "但里面的东西，比电影更科幻。"

旁白: "几十台服务器。十几个人。凌晨三点，没人说话，只有键盘声和空调的嗡嗡声。"

旁白: "那一刻我明白了——这不是小打小闹。这是军队。"

韩默(推开门，声音平淡): "欢迎来到深渊。"

**环境音**: 服务器风扇的嗡嗡声，空调的呼呼声，键盘的敲击声——很多键盘同时敲击的声音

**BGM**: 电子合成音开始出现，频率稳定，像机器的心跳。加入了轻微的脉冲效果，营造科技感。

**歌词(竖排右上角)**:
深渊之门
缓缓打开

**画面动态标记**:
[动] 服务器指示灯闪烁，空调出风口的气流，韩默推门的动作

### 片段 04 [01:30-02:10] - 组织结构揭秘

| 属性 | 内容 |
|------|------|
| 时长 | 40秒 |
| 场景 | 室内/深渊据点 |
| 镜头 | 中景→特写→中景 |
| 情绪 | 紧张、揭秘、警示 |

**画面描写**:
[中景] 韩默带着陆鸣穿过工作区。没有人抬头，但这不意味着他们没在看——陆鸣注意到至少有三个人用余光扫过他。

[特写] 墙上贴着一张大纸。组织结构图。用黑色马克笔画成，线条粗犷。

组织结构图特写:
```
         [投资人] —— 标注："白屋"
            |
         [创始人]
         韩默
            |
    +-------+-------+
    |       |       |
 [核心] [外围] [外围]
 成员    成员    成员
```

[特写] 陆鸣的眼睛。他的瞳孔在"投资人"和"白屋"几个字上停留了很久。

[中景] 韩默站在陆鸣身边，逆光，看不清表情。

**配音/台词**:
旁白: "深渊有三个层级。外围成员负责打杂，外围核心负责技术，核心成员……负责决策。"

旁白: "但最上面，不是韩默。"

韩默(声音平静): "深渊不是我一个人的。"

旁白: "韩默说这句话的时候，没有看墙上的图。他在看我。"

**环境音**: 服务器的嗡鸣声，键盘敲击声，空调的呼呼声

**BGM**: 弦乐开始渐入，不和谐的三和弦，像某种警告。

**歌词(竖排右上角)**:
金字塔的
顶端

**画面动态标记**:
[动] 组织结构图在光线下微微晃动，陆鸣的眼神在结构图上下游移

### 片段 05 [02:10-02:30] - 投资人

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室内/深渊据点/视频会议 |
| 镜头 | 特写→特写 |
| 情绪 | 恐惧、疑惑、毛骨悚然 |

**画面描写**:
[中景] 韩默突然停下脚步。他从口袋里掏出手机，看了一眼屏幕。

[特写] 手机屏幕。一个视频通话请求，对方ID是一串乱码。

[中景] 韩默走到角落，接通电话。他按下免提，举起手机。

[特写] 手机屏幕。逆光。只能看到一个黑色的轮廓，像剪影。背景是一片漆黑，连服务器的绿光都没有。

[特写] 韩默的脸。他的表情变了——不是恐惧，是一种更深的东西。像是……警惕。

**配音/台词**:
投资人(变声器处理过的声音，像金属摩擦): "韩默。新鲜血液？"

韩默(语气平淡): "是。值得培养。"

投资人(停顿了一秒): "好。让他先从外围开始。"

投资人: "别让我失望。"

(通话结束。屏幕暗下。)

[特写] 韩默的脸。他挂断电话后，表情在一瞬间恢复了平静。但陆鸣看到了——在那一秒钟里，他的下颌肌肉紧绷了一下。

**配音/台词**:
旁白: "韩默叫那个人'投资人'。但我看到了他挂断电话后的表情。"

旁白: "那不是金主和打工人的关系。"

旁白: "那是……傀儡和操控者。"

**环境音**: 电话接通的电流声，变声器处理过的电子杂音，电话挂断的提示音

**BGM**: 所有音乐骤停。只剩下变声器那种金属质感的电子音效——像某种不属于人类的声音。然后是寂静。

**歌词(竖排右上角)**:
看不见的
手

**画面动态标记**:
[动] 手机屏幕的微弱光线下，韩默脸部的阴影变化

### 片段 06 [02:30-03:00] - 深渊的真相

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/深渊据点 |
| 镜头 | 中景→特写→正面中景 |
| 情绪 | 严肃、抉择、命运 |

**画面描写**:
[中景] 韩默关掉手机，转向陆鸣。服务器的光在他脸上投下明暗交错的阴影。

[中景] 韩默走向一面墙，墙上投影着一段文字。他按下遥控器，投影亮起。

投影特写:
"深渊三条铁律"
"一、不接伤害普通人的单"
"二、所有行动投票决定"
"三、成员之间不准有私人交易"

[特写] 陆鸣的脸。他的表情是平静的，但眼睛在快速阅读每一条。

[正面中景] 韩默直视陆鸣。

**配音/台词**:
韩默(声音低沉): "深渊不是慈善机构。我们是雇佣兵。"

韩默: "但我们只接该接的单。"

韩默: "你能接受吗？"

旁白: "我没有立刻回答。我在看他的眼睛。"

旁白: "那双眼睛里没有骗人的东西。但也没有全部的真话。"

韩默(直视陆鸣): "这三条，是深渊存在的理由。违反任意一条的人——"

韩默(做了一个抹脖子的动作): "自己消失。"

**环境音**: 投影仪的风扇声，空调的嗡嗡声，韩默和陆鸣的呼吸声

**BGM**: 电子脉冲重新开始，节奏平稳，像某种仪式感。

**歌词(竖排右上角)**:
三条铁律
契约

**画面动态标记**:
[动] 投影文字的光线在墙上微微闪烁，韩默的眼神稳定而锐利

### 片段 07 [03:00-03:20] - 陆鸣的选择

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室内/深渊据点 |
| 镜头 | 特写→中景→特写 |
| 情绪 | 决心、抉择、宿命 |

**画面描写**:
[中景] 陆鸣沉默了很久。他看着那些屏幕上的代码，看着那些沉默工作的人，看着墙上那张组织结构图。

[特写] 陆鸣的眼睛。他的目光最后停留在"白屋"两个字上。

[中景] 陆鸣开口了。他的声音很轻，但很稳。

**配音/台词**:
陆鸣(声音平静): "……我能问一下，那个'投资人'是谁吗？"

[特写] 韩默的表情。他似乎没料到陆鸣会问这个问题。一秒钟的停顿。

[中景] 韩默冷笑了一下。但那笑容里没有恶意，只有一丝……无奈？

韩默: "问太多的人，在深渊活不长。"

韩默(拍了拍陆鸣的肩膀): "但既然你问了——"

韩默: "等你成为核心成员再说。"

[特写] 陆鸣的脸。他没有退缩，没有害怕。他的眼睛里有一种东西在燃烧——不是恐惧，是渴望。

[特写] 陆鸣的手。他攥紧了那张写着"深渊"的纸条。

**配音/台词**:
旁白: "那天晚上，我在深渊的门口站了很久。"

旁白: "我想回头。但我没有。"

旁白: "因为我需要答案。关于爸爸的答案。"

旁白: "深渊可能是深渊。但它也是我当时唯一的线索。"

**环境音**: 服务器的持续嗡鸣，空调的呼呼声，陆鸣的呼吸声

**BGM**: 弦乐渐强，但保持压抑的基调。像黎明前的黑暗。

**歌词(竖排右上角)**:
答案的
代价

**画面动态标记**:
[动] 陆鸣攥紧纸条的手，指节发白
