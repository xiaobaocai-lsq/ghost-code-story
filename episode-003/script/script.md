### 片段 01 [00:00-00:30] - 警察局的长椅

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/警察局 |
| 镜头 | 全景→特写 |
| 情绪 | 失去、迷茫、冰冷 |

**画面描写**:
[全景] 警察局走廊。日光灯惨白，照得一切都像褪色的照片。几张塑料椅子靠墙放着，椅子之间的缝隙像一道道裂缝。

[中景] 陆鸣坐在其中一张椅子上。他的背挺得直直的，双手放在膝盖上，像一个被摆放好的物品。他的脸是灰白的，像一张没有洗干净的纸。

[特写] 陆鸣的眼睛。空洞的，干涩的，像两口被冻住的井。眼睛里没有任何情绪——没有恐惧，没有悲伤，甚至没有困惑。只有一种奇怪的、彻底的平静。

[特写] 一杯热水。有人给他倒的。蒸汽在冷空气中升起，像幽灵。但陆鸣没有碰它。他的眼睛盯着对面的墙壁，一动不动。

[特写] 玻璃窗。上面映出陆鸣的倒影——模糊的，扭曲的，像另一个世界的他。

**配音/台词**:
旁白: "那天晚上，他们把我带到了警察局。"

旁白: "我没有哭，没有说话，也没有发抖。"

旁白: "我只是坐在那里。等着。"

旁白: "我不知道自己在等什么。也许是在等爸爸来接我。"

旁白: "但爸爸不会来了。"

**环境音**: 日光灯的电流嗡嗡声，像蚊子在耳边盘旋。远处有电话铃声，模糊的，像来自另一个世界。偶尔有脚步声经过，但没有人停下来。

**BGM**: 单音钢琴，在一个音符上反复，像心跳在停止前最后的挣扎。极低频的drone开始出现。

**歌词(竖排右上角)**:
等待的
审判

**画面动态标记**:
[静] 陆鸣一动不动，像一尊雕像
[动] 蒸汽缓缓升起，日光灯偶尔闪烁

### 片段 02 [00:30-01:30] - 问话

| 属性 | 内容 |
|------|------|
| 时长 | 60秒 |
| 场景 | 室内/问询室 |
| 镜头 | 中景→特写→中景 |
| 情绪 | 压抑、冷漠、讽刺 |

**画面描写**:
[中景] 问询室。一张桌子，两把椅子。墙上贴着"为人民服务"的标语，但字已经褪色了。

[特写] 表格。一张表格被放在桌上，上面有很多空格等待填写。警察B的手拿着笔，快速地在空格上打勾。

[特写] 陆鸣的脸。他的眼睛盯着桌面，嘴唇紧紧抿着。

警察A(40岁左右，表情麻木): "姓名？"

陆鸣(声音很轻，像从很远的地方传来): "陆鸣。"

警察A: "年龄？"

陆鸣: "八岁。"

警察A: "父亲名字？"

陆鸣停顿了一下。他的眼睛闪了一下，但很快又恢复了空洞。

陆鸣: "陆远航。"

警察B(30岁左右，不耐烦，把笔扔在桌上): "行了行了，这孩子能知道什么？大人打架，小孩在旁边睡觉，醒来爸爸就没了。这种事我见多了。"

警察A叹气。他看了陆鸣一眼，眼神里有一丝——是怜悯吗？但很快就消失了。

警察A: "按程序，'孤儿'要先送收容所。等找到亲属再说。"

[特写] 陆鸣的手。他的手指动了一下——就在听到"孤儿"这个词的时候。但很快就恢复了平静。

**配音/台词**:
警察A: "姓名？"

陆鸣: "陆鸣。"

警察A: "年龄？"

陆鸣: "八岁。"

警察B: "行了行了，这孩子能知道什么？"

警察A: "按程序，'孤儿'要先送收容所。"

旁白: "这是我第一次听到'孤儿'这个词。"

旁白: "他们说这个词的时候，就像在说一件物品。"

旁白: "一件等待处理的物品。"

旁白: "没有人问我：那天晚上你看到了什么？你爸爸最后说了什么？"

旁白: "没有人问。"

**环境音**: 笔在纸上划动的声音，沙沙的。椅子挪动的声音，沉闷的。窗外有风声，像有人在叹息。

**BGM**: 无旋律，只有低频drone，像来自地底的声音。节奏是机械的，像打字机的声音。

**歌词(竖排右上角)**:
程序的
冷漠

**画面动态标记**:
[动] 警察B的笔快速移动，在表格上打勾
[静] 陆鸣一动不动，只有眼睛偶尔眨一下

### 片段 03 [01:30-02:30] - 等待

| 属性 | 内容 |
|------|------|
| 时长 | 60秒 |
| 场景 | 室内/收容所 |
| 镜头 | 全景→中景→特写 |
| 情绪 | 孤独、封闭、空洞 |

**画面描写**:
[全景] 收容所。一个大房间，里面有几张简易床和椅子。墙上贴着儿童画，但颜色已经褪去。窗户很高，只能看到一小块天空。

[中景] 几个孩子。有的在哭，声音沙哑，像被拧干的毛巾。有的在发呆，眼神空洞地望着某个方向。一个小女孩抱着一个破旧的布娃娃，嘴唇在动，像在自言自语。

[特写] 陆鸣。他蹲在一个角落里，背靠着墙。他的眼睛没有看任何人，只是盯着地面。地面是灰色的，有几道裂缝。

[特写] 陆鸣的手。他的手指开始在膝盖上轻轻敲击——一下，两下，三下……

[中景] 一个女人在角落里打电话，声音很大："……对，是孤儿……不，没有亲属……直接送孤儿院就行了……"

[特写] 陆鸣的脸。他的表情没有任何变化。他听到了，但他没有任何反应。他只是继续数着。

**配音/台词**:
旁白: "收容所里有很多孩子。"

旁白: "他们有的在哭，有的在发呆。"

旁白: "我谁都没有看。我只是数着地板上的格子。"

旁白: "一、二、三、四……"

旁白: "数到一百，又从头开始。"

旁白: "数字让我安静。"

旁白: "数字不会问问题。"

旁白: "数字不会说谎。"

**环境音**: 孩子的哭声，模糊的，远处的。女人的电话声，清晰的，刺耳的。有人在翻东西，哗啦哗啦的。风声从窗缝钻进来，发出呜咽。

**BGM**: 钢琴声开始出现，音符稀疏，像雨滴落在空旷的大厅。节奏很慢，像心跳在减速。

**歌词(竖排右上角)**:
数字成为
庇护所

**画面动态标记**:
[动] 陆鸣的手指在膝盖上轻轻敲击，节奏稳定
[动] 窗外的小块天空在云层中若隐若现

| 镜头 | 特写快切 |

**画面描写**:
[特写] 天花板。36块瓷砖，整整齐齐地排列着。

[特写] 陆鸣的眼睛。他的眼睛在追踪每一块瓷砖，像在执行某种任务。

[快切] 窗户。12根栏杆，锈迹斑斑。

[快切] 裂缝。从墙角开始，弯弯曲曲，长度大约等于陆鸣的前臂。

[快切] 陆鸣的手。他的手指在膝盖上敲击，节奏稳定得像节拍器。

[特写] 陆鸣的脸。他的表情是空洞的，但眼睛深处有什么在燃烧——不是悲伤，不是愤怒，是一种奇怪的、专注的光芒。

**配音/台词**:

旁白: "天花板有36块瓷砖。"

旁白: "那道裂缝大约是我的前臂那么长。"

旁白: "数数的时候，我什么都不用想。"

旁白: "数数的时候，我是安全的。"

旁白: "数字是诚实的。"

旁白: "数字不会骗我。"

**环境音**: 陆鸣手指敲击膝盖的声音，稳定的，轻微的。远处有孩子仍在哭，但声音变得更远了，像隔着一层玻璃。

**BGM**: 节拍器般的节奏，机械的，冷酷的。钢琴声完全消失，只剩下极低频的drone。

**歌词(竖排右上角)**:

### 片段 05 [03:00-03:20] - 被遗忘的孩子

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室内/收容所 |
| 镜头 | 中景→特写 |
| 情绪 | 接受、宿命、空洞 |

**画面描写**:
[中景] 收容所。阳光从窗户照进来，在地上投下一块光斑。但那块阳光是冷的，照不到陆鸣蹲着的角落。

[中景] 两个警察在窗边说话，背对着陆鸣。

警察A: "这孩子没有亲属。"

警察B: "那就按程序走。郊区孤儿院有空位。"

[特写] 陆鸣的脸。他的眼睛望着窗户方向，但没有焦距。他听到了，但他没有任何反应。他只是继续数着——或者，已经不数了，只是沉浸在某种空洞中。

[特写] 一个牛皮纸档案袋特写。上面写着"陆鸣"两个字。然后是一个红色的印章："孤儿"。下面还有一行字："待处理"。

**配音/台词**:
警察A: "这孩子没有亲属。"

警察B: "那就按程序走。郊区孤儿院有空位。"

旁白: "他们说'按程序走'。"

旁白: "程序。Process。过程。"

旁白: "一个词，就把我的一生决定了。"

旁白: "没有人问我愿不愿意。"

旁白: "没有人问我想去哪里。"

旁白: "那天，我学会了一件事。"

旁白: "在这个世界上，有些词可以定义你。"

旁白: "孤儿。这就是我。"

**环境音**: 脚步声渐渐远去。门被打开又关上。收容所变得更安静了，只剩下日光灯的嗡嗡声。

**BGM**: 弦乐开始出现，低沉的，像远处的雷声。但不是愤怒，是接受——一种彻底的、宿命般的接受。

**歌词(竖排右上角)**:
孤儿
这个词

**画面动态标记**:
[动] 档案袋上的红色印章在光线下闪烁
[静] 陆鸣的眼睛——已经完全空洞了
