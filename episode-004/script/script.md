| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室内/收容所 |
| 镜头 | 全景→特写 |
| 情绪 | 空洞、平静、冰冷 |

**画面描写**:
[全景] 收容所的早晨。阳光从高窗户斜斜地照进来，在地上投下几道光柱。光柱里有灰尘在浮动——细小的，密密麻麻的，像某种微小的生命。

[中景] 陆鸣蹲在角落里，背靠着墙。他的眼睛追随着阳光中漂浮的灰尘，嘴唇在轻微地动着。

[特写] 陆鸣的眼睛。他的眼睛里有一种奇怪的专注——不是悲伤，不是愤怒，只是一种空洞的、能穿透一切的凝视。

[特写] 阳光中的灰尘。镜头拉近，灰尘在光线中翻滚，像无数微小的世界。

**配音/台词**:

旁白: "收容所的窗户很高，阳光从那里照进来的时候，我能看见灰尘。"

旁白: "我在数灰尘。"

旁白: "一、二、三、四……"

**环境音**: 收容所的安静，远处有孩子的声音，模糊的。风声从窗缝钻进来，发出细微的呜咽。

**BGM**: 单音钢琴，在一个音符上反复，像某种机械的心跳。

**歌词(竖排右上角)**:

**画面动态标记**:

| 属性 | 内容 |
|------|------|

| 场景 | 室内/收容所 |
| 镜头 | 中景→特写 |
| 情绪 | 压抑、震惊、困惑 |

**画面描写**:
[中景] 收容所的办公区。一个工作人员坐在桌子后面打电话，声音透过嘈杂的空气传过来。

[特写] 电话听筒。工作人员的声音断断续续地传来。

工作人员: "……对，那个陆远航的案子……结了……经济纠纷……"

[特写] 陆鸣的脸。他的眼睛动了一下——就在听到"陆远航"这个名字的时候。但很快又恢复了平静。

工作人员(画外音): "……三天就结了……证据不足……"

[特写] 陆鸣的手。他的手指开始敲击膝盖——一、二、三、四……节奏变得更慢了，像是在数什么东西。

工作人员(画外音): "……没亲属……直接送孤儿院……"

[特写] 陆鸣的眼睛。他的眼睛望着地面，嘴唇在动。

**配音/台词**:

旁白: "他们说是'经济纠纷'。"

旁白: "三天。"

旁白: "从爸爸死到结案，三天。"

旁白: "七十二小时。"

旁白: "四千三百二十分钟。"

旁白: "二十五万九千二百秒。"

旁白: "这是我记住的第一个数字。"

**环境音**: 电话的嘟嘟声，工作人员的声音模糊成背景噪音。收容所里的嘈杂声像隔着一层水。

**BGM**: 低频drone开始出现，像远处的雷声。节奏是机械的，像打字机。

**歌词(竖排右上角)**:
七十二
小时

**画面动态标记**:
[动] 陆鸣的手指在膝盖上敲击，节奏越来越慢
[静] 他的脸完全没有表情

[中景] 陆鸣蹲在角落里，阳光照不到他。他的脸在阴影中，只有眼睛在闪烁。

**配音/台词**:

旁白: "我不懂。"

旁白: "'快速结案'。"

**环境音**: 收容所的嘈杂声变得更远了，像隔着一层玻璃。陆鸣的呼吸声，轻微的，稳定的。

**歌词(竖排右上角)**:

审判

[静] 陆鸣的眼睛一动不动
