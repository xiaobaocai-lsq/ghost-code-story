# 幽灵代码 (Ghost Code)

一部AI漫剧，悬疑犯罪类型，200话超长篇。

## 故事简介

1993年冬天，8岁的陆鸣亲眼目睹父亲被灭门，被送进孤儿院。13岁时逃出孤儿院，在网吧结识黑客组织"深渊"，走上犯罪道路。17岁被网警逮捕，被迫成为线人"幽灵"。在黑暗与光明的夹缝中，他必须面对父亲死亡的真相，找到自己的救赎之路。

## 主角

- **陆鸣/幽灵** - 童年遭受巨大不幸的孤儿，黑客天赋极高，在犯罪与正义之间挣扎

## 关键词

- #悬疑 #黑客 #犯罪 #救赎 #90年代 #信息安全 #时代背景

## 故事大纲结构

- 第一幕：黑暗起源（1-30话）
- 第二幕：深渊沉沦（31-80话）
- 第三幕：至暗时刻（81-110话）
- 第四幕：双面幽灵（111-160话）
- 第五幕：幽灵归来（161-200话）

## 项目结构

```
ghost-code-story/

├── plots/                # 分章节剧情

```
