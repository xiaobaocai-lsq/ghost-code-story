| 年龄段 | 描述 |

## 核心创伤

**8岁那年**：亲眼目睹父亲陆远航被杀害，躲在衣柜里从缝隙看着一切。

**关键记忆碎片**：
- 父亲临死前的话："不是我...是他们..."
- 凶手手腕上有一块表，表盘是白色的
- 父亲最后看向衣柜的方向——他知道陆鸣在里面

## 心理成长曲线

```
8岁     破碎期   亲眼目睹灭门        情感解离       失去"哭"的能力
8-13岁  冻结期   孤儿院冷暴力        攻击性自闭     用数字逃避现实
13-17岁 伪装期   深渊组织"家"的假象   角色扮演       成为"幽灵"
17岁    撕裂期   被迫成为线人        身份分裂       真我/幽灵双重人格
17-20岁 黑暗期   师父背叛、院长被杀   复仇执念       人性被黑暗吞噬
20-22岁 虚无期   发现自己不过是棋子   存在主义危机   失去所有信任
22-25岁 重建期   被追杀中的意外温暖   选择性信任     学会依靠他人
25岁    觉醒期   最终对决            接受牺牲       找回"陆鸣"的名字
```

## 性格特征

### 优点
- 极强的适应能力
- 对弱者有本能的保护欲
- 一旦信任就绝对忠诚
- 在压力下异常冷静

### 缺点
- 不擅长表达情感
- 过度自我牺牲
- 难以建立信任
- 对"家人"有执念

---

## 价值观演变

### 前期（认同深渊信仰）
```
"代码即自由，信息应该流动，不该有秘密"
```

### 中期（开始怀疑）
```
"如果自由被用来伤害无辜的人呢？"
```

### 后期（形成自己的信仰）
```
"真正的自由不是没有边界，而是在边界内仍然选择善良"
```

## 人物关系

### 父亲：陆远航（已故）
- 1970-1993
- 网络安全工程师，被卷入"长城公司案"
- 死前试图揭露真相，是白屋的早期成员

### 师父：韩默（已故）
- 原深渊组织创始人
- 对陆鸣亦师亦父
- 真实身份是陆远航的线人

### 联络人/初恋：苏晴
- 网络安全部门警员
- 在陆鸣最黑暗的时候出现
- 代表着"外面的世界"

### 导师：陈Sir
- 老警察
- 陆远航的旧友
- 用自己的方式赎罪
