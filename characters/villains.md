## 1. "棋手" —— 韩默

### 基本信息

| 项目 | 内容 |
|------|------|
| 代号 | 棋手 |
| 本名 | 韩默（原名已抹除） |
| 年龄 | 1960年生 |
| 身份 | 深渊组织创始人，陆鸣师父 |

### 外貌

- 消瘦儒雅
- 永远西装革履
- 45岁看起来像35
- 有一双修长的手，擅长弹钢琴

### 性格特征

- 极度理性，把一切都当作"棋局"
- 对"纯粹技术"有偏执
- 瞧不起"dirty hands"
- 内心深处有未愈合的伤

### 弱点

- 无法放下过去
- 对陆鸣产生了真实的感情
- 在最后选择了赎罪

### 完整背景故事

```
1978年 - 18岁，全国青少年计算机冠军，被国防大学特招
1982年 - 爱上同班同学林薇，理想主义的最后一年
1985年 - 林薇因"政治问题"被带走调查，韩默被逼配合"网络安全项目"
1987年 - 林薇狱中病死，韩默彻底黑化，加入白屋前身
1990年 - 用技术换取资源，建立"深渊"作为白屋的"白手套"
1998年 - 遇见陆鸣，看到年轻时的自己，决定"改造"而非"培养"
2005年 - 白屋要求处理"陆远航的儿子"，韩默第一次动摇
2010年 - 开始秘密收集白屋罪证，发现自己已被监控
2020年 - 陆鸣卧底身份曝光，韩默选择用命换陆鸣一条生路
```

### 核心矛盾

恨体制恨到成为体制的帮凶，又恨自己恨到想死

### 经典场面

- 第79话：韩默教陆鸣下棋，"记住，落子无悔，但棋盘是你的"
- 第152话：韩默之死——"我这辈子只下过一盘好棋，就是你"

### 台词

```
"代码即自由，但那是我年轻时候的事了"
"你太像他了，所以我必须毁掉你"
"我这辈子只下过一盘好棋，就是你"
```

## 2. "白屋"幕后Boss —— "先知"

### 基本信息

| 项目 | 内容 |
|------|------|
| 代号 | 先知 / The Prophet |
| 身份 | 白屋最高决策者 |
| 真相 | 可能是一个人，也可能是一个委员会 |

### 设计理念

**终极反派不是恶人，是"为了更大的善"的知识分子精英**

```
"不是我要控制这个世界，是这个世界需要一个方向盘"
```

### 目的

- 1980年代就预见"信息时代权力转移"，开始布局
- 通过控制信息来控制未来
- 相信精英应该统治普通人

### 标志

- 每次重大网络事件后，都会收到一朵白玫瑰

### 关键决策（幕后影响）

| 年份 | 决策 |
|------|------|
| 1993年 | 批准处理陆远航（知道太多） |
| 2005年 | 批准深渊组织的存在（脏手套） |
| 2020年 | 批准"天网计划"（金融系统后门） |
| 2030年（结局） | 计划被陆鸣阻止，但"先知"早已转移 |

### 悬念

先知究竟是谁？
- 暗示1：陈Sir暗示"可能在最高层"
- 暗示2：韩默死前说"他比我们都老，是一代人的阴谋"
- 暗示3：苏晴父亲临死前提到"老同学，你不该回来的"

## 3. "灰烬" —— 裴婉

### 基本信息

| 项目 | 内容 |
|------|------|
| 代号 | 灰烬 / Ash |
| 年龄 | 1988年生 |
| 职业 | 白屋执行者，女性冷酷杀手 |
| 外貌 | 冷艳，职业杀手外表下的高学历女性 |

### 性格特征

- 极度冷静，非必要不说话
- 把杀人当作"工作"
- 不知道自己想要什么
- 被"先知"用存在危机操控

### 天赋

- 伪装
- 渗透
- 心理战
- 曾是国安系统精英

### 独立成长线

```
2005年 - 17岁，被"白屋"从孤儿院带走培养
2010年 - 成为最优秀的执行者，从不质疑命令
2015年 - 第一次动摇：目标是个有孩子的母亲，她选择伪造报告
2018年 - 被"先知"点名负责追踪"幽灵"
2020年 - 与陆鸣多次交锋，发现他"为什么而活"的答案让她崩溃
2022年 - 在一次任务中救了陆鸣一命，"你欠我一次"
2025年 - 最后选择背叛白屋，为陆鸣挡子弹
```

### 与陆鸣的关系线

- 最纯粹的"猫鼠游戏"，却产生奇异的理解
- 裴婉："我不是来杀你的，我是来确认你还活着"
- 最后的救赎：用自己的死亡证明"我不是工具"

### 经典场面

- 第138话：裴婉第一次放过陆鸣，"你今天死了，我的任务就完了"
- 第179话：裴婉的真实目的曝光——她想亲眼看到"幽灵"能走多远
- 第195话：裴婉之死，"先知让我杀你，但我想看你赢一次"

### 台词

```
"我不是来杀你的，我是来确认你还活着"
"你欠我一次"
"先知让我杀你，但我想看你赢一次"
```

## 4. "棋圣" —— 师祖

### 基本信息

| 项目 | 内容 |
|------|------|
| 代号 | 棋圣 / Master |
| 年龄 | 1945年生，70多岁 |
| 身份 | 深渊组织的"师祖"，白屋的元老 |

### 外貌

- 精神矍铄，永远在打太极
- 看起来像个普通退休老人
- 说话永远慢条斯理

### 性格特征

- 极度耐心，只说模棱两可的话
- 让你自己理解
- 真正的操控者

### 背景

- 80年代"信息监控"项目发起人之一
- 韩默的师父
- "白屋"早期真正的策划者

### 关键台词

```
"默儿，你太急了，棋要慢慢下，要下到第三代人的棋盘上"
"我下棋从不是为了赢，是为了没有人能赢"
```

### 设计意图

终极反派不是"坏人"，是"相信自己在替天行道"的精英

## 5. "蜃" —— 镜像

### 基本信息

| 项目 | 内容 |
|------|------|
| 代号 | 蜃 / Mirage |
| 身份 | 网络身份专家 |
| 性别 | 不明，永远用变声器 |
| 隶属 | 深渊组织技术核心，Needle的师父 |

### 性格特征

- 极度偏执
- 只信任代码，认为"人"是不可靠的
- 真正的恐惧：被遗忘
- 所以创造无数分身

### 天赋

- 网络身份伪造
- 创造了几十个不存在的"人"
- 幽灵人口创造者

### 独立成长线

```
1995年 - 因"网络谣言"入狱一年，出狱后彻底匿名化
2000年 - 成为最顶尖的"幽灵人口"创造者
2010年 - 收了Needle为徒，教会他"代码即身份"
2015年 - Needle背叛后，"蜃"选择沉默，不再信任任何人
2020年 - 被迫参与"天网计划"，但留了后门
2025年 - 与陆鸣秘密接触，"我可以给你真名，但你得答应我一件事"
```

### 经典场面

- 第165话："蜃"第一次露面（声音），"我不是来帮你的，我只是讨厌蜃楼"

### 台词

```
"在网络上，没有人知道你是一条狗"
"我不是来帮你的，我只是讨厌蜃楼"
```

## 6. 苏建国 —— "岳父"

### 基本信息

| 项目 | 内容 |
|------|------|
| 身份 | 苏晴的父亲 |
| 年龄 | 1945-1950年生 |
| 真实身份 | 白屋核心成员，"先知"的得力助手 |

### 背景

- 陆远航的大学室友
- 一起加入白屋
- 在陆远航试图退出时，被迫签字同意"处理"

### 核心矛盾

"我不是坏人，我只是比你爸更现实"

### 结局

- 第182-192话：真实身份揭露
- 苏晴开枪打伤自己的父亲
- 苏建国自杀，死前留下忏悔信

### 设计意图

展示一个"好人"如何一步步变成"恶魔"，以及对家人的伤害
