# 正派配角

## 1. 苏晴 —— "灯塔"

### 基本信息

| 项目 | 内容 |
|------|------|
| 代号 | 灯塔 |
| 年龄 | 与陆鸣相仿或略大 |
| 职业 | 网络安全部门警员，后成为金融监管顾问 |
| 家庭背景 | 法律世家，父亲苏建国是老一辈网络安全专家 |

### 性格特征

- 正义感极强但不懂灰色地带
- 天真型理想主义者
- 说话直接，不懂得拐弯
- 内心柔软但外表坚硬

### 天赋

- 社会工程学（人肉搜索、心理学操控）
- 审讯技巧
- 非技术出身，但后天努力学习

### 弱点

- 无法理解陆鸣的黑暗
- 用"正确"伤害他
- 在感情上迟钝

### 独立成长线

```
19岁 - 警校优秀毕业生，拒绝父亲安排的升迁，选择网警基层
21岁 - 第一次办案遇到陆鸣，被他的"黑暗"吸引又恐惧
23岁 - 发现父亲与"白屋"旧事有关，信仰崩塌
24岁 - 质问父亲，父亲自杀未遂，揭示：为打入白屋内部，不得已沉默多年
25岁 - 理解陆鸣的选择，开始在体制内做"灰色任务"
```

### 与陆鸣的关系线

| 阶段 | 描述 |
|------|------|
| 初期 | 觉得他是个"该死的混蛋但该死的有魅力" |
| 中期 | 发现他内心比她还孤独，产生救赎欲 |
| 后期 | 发现她父亲和陆鸣父亲的命运纠缠，决定"一起赎罪" |

### 经典场面

- 第95话：审讯室对峙，陆鸣说"你不该来救我这种人"，她回答"我就该来"
- 第148话：陆鸣身份曝光，苏晴顶住压力放他走，"我会在出口等你"
- 第198话：陆鸣失踪后，她用十年追查"幽灵"的下落

## 2. 老黑 —— "锚"

### 基本信息

| 项目 | 内容 |
|------|------|
| 代号 | 老黑 |
| 本名 | 刘建国 |
| 年龄 | 1975年生（比陆鸣大约10岁） |
| 职业 | 前银行技术主管，因"挪用公款"入狱 |
| 家庭背景 | 农村出身，拼命读书改变命运 |

### 性格特征

- 沉默寡言但极度忠诚
- 经历过真正的社会毒打
- 对"家"有执念
- 讲义气，重承诺

### 天赋

- 硬件破解、嵌入式系统
- 服务器架构
- 是"幽灵"的技术备份

### 独立成长线

```
1998年 - 22岁，发现银行系统漏洞，被威胁成为帮凶
2000年 - 为保护家人，独自扛下罪名入狱
2005年 - 狱中遇见同样被冤枉的孤儿院孩子，触发恻隐之心
2010年 - 出狱，发现家人已搬走，"消失"是最好的保护
2013年 - 遇见逃亡中的陆鸣，看到年轻时的自己
```

### 与陆鸣的关系线

- 从"凭什么救我"到"我欠你命，你欠我义"
- 不是兄弟情，是"我欠你命，你欠我义"的纠缠

### 经典场面

- 第157话：老黑在陆鸣快死时说"我帮你扛一次，这次轮到我"
- 第175话：老黑为保护陆鸣，主动暴露自己，"幽灵得有人守，老黑我来当"

## 3. Needle —— "镜"

### 基本信息

| 项目 | 内容 |
|------|------|
| 代号 | Needle |
| 背景 | 香港，混迹东南亚黑客圈 |
| 年龄 | 1994年生（比陆鸣小） |
| 职业 | 自由黑客，漏洞猎人 |

### 性格特征

- 极度聪明但情绪不稳定
- 渴望"被看见"
- 玩世不恭的外表下是深深的伤痛
- 对陆鸣有复杂的情感

### 天赋

- 漏洞挖掘
- 创造了多个0day exploit
- 网络对战

### 独立成长线

```
2008年 - 14岁，香港黑客比赛冠军，被安全局盯上
2010年 - 被招募又背叛，成为自由黑客
2012年 - 在"深渊"短暂时期，与陆鸣有过网络对决
2015年 - 爱人被跨国黑帮杀害，用技术复仇，从此只认钱
2020年 - 被陆鸣"感化"（其实是交易），成为外围协助者
```

### 与陆鸣的关系线

- 网络上打过无数次的对手，却从未见面
- "你是另一个版本的我，如果我那年没有失去她"
- 最终用自己的死亡为陆鸣争取时间

### 经典场面

- 第133话：线上对决，激战三天三夜后，陆鸣突然停手："你不该是这样的"
- 第188话：Needle临死前："幽灵，帮我看看...太阳..."

## 4. 陈Sir —— "影"

### 基本信息

| 项目 | 内容 |
|------|------|
| 代号 | 影 |
| 本名 | 陈志远 |
| 年龄 | 1965年生（快50的老警察） |
| 职业 | 网络安全部门元老 |
| 家庭背景 | 已离婚，前妻刘姐是护士 |

### 性格特征

- 表面圆滑世故，内心坚持程序正义
- 80年代大学生，亲历理想主义幻灭
- 用"灰色"手段做"白色"的事
- 有深深的愧疚感

### 独立成长线

```
1988年 - 23岁，刚入职，奉命调查陆远航案，良心不安
1993年 - 眼睁睁看着陆远航被定罪，无能为力
2005年 - 成为网络安全部门元老，用自己的方式"赎罪"
2010年 - 主导"招安"陆鸣计划，试图用他扳倒白屋
2020年 - 被上级施压，必须交出陆鸣，选择放走他
```

### 与陆鸣的关系线

| 阶段 | 描述 |
|------|------|
| 初期 | 利用关系，各取所需 |
| 中期 | 发现陆鸣身上有"老友"的影子，产生父辈的愧疚 |
| 后期 | 用自己的退休/牺牲换取陆鸣的生存机会 |

### 经典场面

- 第98话：陈Sir私下告诉陆鸣真相，"我没有资格求你原谅"
- 第154话：陈Sir最后出现在陆鸣逃亡现场，假装没看见他
