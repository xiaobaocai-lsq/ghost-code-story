### 片段 01 [00:00-00:20] - 一个月后

| 属性 | 内容 |
|------|------|
| 时长 | 20秒 |
| 场景 | 室内/网吧 |
| 镜头 | 中景→特写 |
| 情绪 | 平静、等待、暗流涌动 |

**关键帧规划**:
- 00:00-00:05: 网吧日常
- 00:05-00:10: 陆鸣工作
- 00:10-00:15: 陆鸣的内心
- 00:15-00:20: 等待的感觉

**画面描写**:
[中景] 畅通网吧。白天，客人不多。陆鸣在整理键盘，动作很熟练。

[特写] 陆鸣的脸。一年过去了，他瘦了一些，但眼神比以前更锐利。

[特写] 陆鸣的手。稳定，有力，不再像以前那样颤抖。

旁白: "一个月后，我的日常生活没有变化。"

旁白: "白天在网吧打工，晚上睡在杂物间。"

旁白: "王奶奶还是每天早上五点起来给我热牛奶。"

旁白: "但我知道，有什么东西在等待。"

[特写] 陆鸣望向窗外。阳光很亮，他的眼睛眯了起来。

旁白: "深渊没有联系我。"

旁白: "但我知道他们一直在看着。"

**配音/台词**:
旁白: "一个月后，我的日常生活没有变化。白天在网吧打工，晚上睡在杂物间。但我知道，有什么东西在等待。"

**环境音**: 网吧的嘈杂声，键盘声，窗外的鸟叫声

**BGM**: 钢琴声平静但带有暗流，像平静的水面下有东西在游动。

**歌词(竖排右上角)**:
等待
有时候
比行动
更难

| 镜头 | 中景→特写 |

- 00:30-00:40: 韩默观察
- 00:40-00:50: 韩默看向陆鸣

[中景] 网吧门口。阳光从门外照进来，逆光中一个人影走进来。

[中景] 韩默走进来。他穿着灰色的外套，很普通，像是任何一个路人。但他的步伐很稳，每一步都踩得很准。

[特写] 韩默的脸。35岁左右，消瘦，颧骨很高。眼镜后面是一双深邃的眼睛——像两口深井，看不见底。

### 片段 03 [00:50-01:20] - 你的眼睛像我认识的一个人

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/网吧→室外 |
| 镜头 | 特写→中景 |
| 情绪 | 震惊、紧张、心跳加速 |

**关键帧规划**:
- 00:50-01:00: 韩默盯着陆鸣
- 01:00-01:10: 韩默说出那句话
- 01:10-01:20: 陆鸣的反应

**画面描写**:
[中景] 韩默走向陆鸣。他的脚步很轻，但每一步都像踩在陆鸣的心脏上。

[特写] 韩默的眼睛。他盯着陆鸣的脸看了很久，像是在找什么东西。

[特写] 陆鸣的脸。他的呼吸停了一秒。

韩默: "跟我来。"

旁白: "他没有问我愿不愿意。"

旁白: "他直接转身往外走。"

旁白: "我跟了上去。"

[中景] 韩默走出网吧，陆鸣跟在后面。王奶奶从柜台后面抬起头，看了看他们，但没有说话。

[远景] 两人走在小巷里。韩默走在前面，陆鸣跟在后面。阳光被建筑物挡住，小巷里很暗。

[中景] 韩默突然停下脚步。他转过身，看着陆鸣。

韩默: "你的眼睛。"

陆鸣: "……？"

韩默: "像我认识的一个人。"

[特写] 陆鸣的心跳漏了一拍。他的手不自觉地攥紧了。

旁白: "我的心跳漏了一拍。"

旁白: "像认识的一个人。"

旁白: "谁？"

旁白: "我的父亲？他们认识？"

**配音/台词**:
韩默: "跟我来。"
韩默: "你的眼睛。像我认识的一个人。"
旁白: "我的心跳漏了一拍。"

**环境音**: 脚步声停止，沉默

**BGM**: 钢琴单音，像心跳停止的那一拍。

**歌词(竖排右上角)**:
像我认识的
一个人
我的父亲
他们认识？

- 01:30-01:40: 韩默回避
- 01:40-01:50: 陆鸣父亲的问题

**画面描写**:

陆鸣: "谁？"

韩默: "很久以前的事了。"

[特写] 韩默的眼神。他回避了陆鸣的目光，看向别处。

韩默: "你父亲叫什么名字？"

[特写] 陆鸣的身体。他的身体僵了一下。

陆鸣: "……陆远航。"

[特写] 韩默的脸。听到这个名字，他的表情微微变了一下。只有一瞬间，然后恢复了平静。

韩默: "我知道。"

[特写] 陆鸣的眼睛。他的眼睛里有什么东西在燃烧。

陆鸣: "你认识他？"

韩默: "很久以前。"

韩默: "现在不是谈这个的时候。"

旁白: "他知道。"

旁白: "他认识我父亲。"

旁白: "但他不肯说。"

[中景] 韩默转身继续走。他的背影在暗巷里显得很瘦削。

韩默: "跟上。"

韩默: "很久以前的事了。你父亲叫什么名字？"
陆鸣: "陆远航。"
韩默: "我知道。"
陆鸣: "你认识他？"
韩默: "很久以前。现在不是谈这个的时候。"
旁白: "他知道。他认识我父亲。但他不肯说。"

**环境音**: 沉默，脚步声

**BGM**: 低沉的弦乐继续，像压抑的情绪在聚集。

|------|------|

| 场景 | 室内/小房间 |
| 镜头 | 中景 |
| 情绪 | 专注、好奇、期待 |

**关键帧规划**:
- 01:50-02:00: 到达小房间
- 02:00-02:10: 房间内景
- 02:10-02:20: 韩默开始说教

**画面描写**:
[中景] 一栋旧楼的门口。韩默推开门，带陆鸣走进去。

[中景] 小房间。很旧的水泥墙，一张桌子，一把椅子，一台电脑。窗户很小，光线很暗。

[中景] 韩默坐在椅子上。他示意陆鸣也坐下。

韩默: "你知道黑客最重要的是什么？"

[特写] 陆鸣的脸。他在思考。

陆鸣: "技术？"

韩默: "不是。"

韩默: "是耐心。"

旁白: "韩默带我去了一栋旧楼。"

旁白: "一个小房间，里面只有一台电脑。"

韩默: "破解不是比谁快。是比谁不出错。"

韩默: "急躁的人，永远成为不了真正的黑客。"

[特写] 陆鸣的眼睛。他在记住每一个字。

**配音/台词**:
旁白: "韩默带我去了一栋旧楼。有一个小房间，里面只有一台电脑。"
韩默: "你知道黑客最重要的是什么？陆鸣：技术？韩默：不是。是耐心。破解不是比谁快。是比谁不出错。急躁的人，永远成为不了真正的黑客。"

**环境音**: 电脑风扇声，寂静

**BGM**: 钢琴声变得缓慢而深沉，像传授智慧的时刻。

**歌词(竖排右上角)**:

### 片段 06 [02:20-02:50] - 第一课结束

| 属性 | 内容 |
|------|------|
| 时长 | 30秒 |
| 场景 | 室内/小房间 |
| 镜头 | 特写 |
| 情绪 | 决心、期待、新的开始 |

**关键帧规划**:
- 02:20-02:30: 韩默留下任务
- 02:30-02:40: 陆鸣看着任务
- 02:40-02:50: 韩默离开

**画面描写**:
[中景] 韩默站起来。他走到电脑前，打开一个文件。

[特写] 屏幕上显示着一行字："第一课：破解FireWall v3.2"

韩默: "这是你的第一课。"

韩默: "明天这个时候，我会来。"

[特写] 陆鸣的脸。他的眼神里有决心。

韩默: "如果你没有破解，说明你不适合这条路。"

韩默: "如果你破解了——"

韩默: "我们再谈。"

[中景] 韩默走向门口。他在门边停下，回头看了陆鸣一眼。

韩默: "陆鸣。"

陆鸣: "嗯？"

韩默: "你父亲的事，我会告诉你的。"

韩默: "但不是现在。"

[特写] 韩默的眼睛。那眼神里有一种复杂的情绪。

韩默: "等你值得知道的时候。"

[中景] 韩默离开。门关上了。

[特写] 陆鸣看着关上的门，然后转向电脑屏幕。

旁白: "他走了。"

旁白: "我看着桌上的任务——破解一个防火墙。"

旁白: "FireWall v3.2。"

旁白: "我知道，真正的训练开始了。"

旁白: "我一定要破解它。"

旁白: "为了知道父亲的事。"

旁白: "也为了知道——我是谁。"

韩默: "这是你的第一课。明天这个时候，我会来。如果你没有破解，说明你不适合这条路。如果你破解了——我们再谈。"
韩默: "陆鸣。你父亲的事，我会告诉你的。但不是现在。等你值得知道的时候。"
旁白: "我知道，真正的训练开始了。我一定要破解它。为了知道父亲的事。也为了知道——我是谁。

**环境音**: 门关上，然后是电脑风扇声

**BGM**: 钢琴声渐强，像决心的主题在升起。

**歌词(竖排右上角)**:

父亲的事
也为了知道
